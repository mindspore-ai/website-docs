std::shared_ptr<Session> MSPREDICT_API mindspore::predict::CreateSession ( const char *  graphBuf,
size_t  size,
const Context &  ctx 
)

MindSpore predict neural network session create function.

This function used to create MindSpore predict neural network session, which will be used to run the neural network.

Parameters
[in]sessionNameThe name of the session.
[in]graphBufThe buffer of the graph, used for build session.
[in]sizeThe size of the session.
[in]ctxThe context of the session.
Returns
Instance of MindSpore predict session.
Note
The caller needs to allocate and free memory of graph buffer.
