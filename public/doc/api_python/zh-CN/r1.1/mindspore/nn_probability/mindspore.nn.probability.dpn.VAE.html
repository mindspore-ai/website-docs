Variational Auto-Encoder (VAE).

The VAE defines a generative model, Z is sampled from the prior, then used to reconstruct X by a decoder. For more details, refer to Auto-Encoding Variational Bayes.

Note

When the encoder and decoder are defined, the shape of the encoder‚Äôs output tensor and decoder‚Äôs input tensor must be \((N, hidden\_size)\). The latent_size must be less than or equal to the hidden_size.

  • decoder (Cell) ‚Äì The DNN model defined as decoder.

  • hidden_size (int) ‚Äì The size of encoder‚Äôs output tensor.

  • latent_size (int) ‚Äì The size of the latent space.
