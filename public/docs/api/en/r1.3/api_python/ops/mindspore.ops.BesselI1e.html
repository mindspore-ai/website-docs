mindspore.ops.BesselI1e

class mindspore.ops.BesselI1e(*args, **kwargs)[source]

Computes BesselI1e of input element-wise.

Inputs:

  • x (Tensor) - The shape of tensor is \((x_1, x_2, ..., x_R)\). Data type must be float16 or float32.

Outputs:

Tensor, has the same shape as x.

Raises

TypeError – If x is not a Tensor.

Supported Platforms:

Ascend

Examples

>>> bessel_i1e = ops.BesselI1e()
>>> x = Tensor(np.array([0.24, 0.83, 0.31, 0.09]), mindspore.float32)
>>> output = bessel_i1e(x)
>>> print(output)
[0.09507662 0.19699717 0.11505538 0.04116856]
