class mindspore.nn.ResizeBilinear[source]ÔÉÅ

Samples the input tensor to the given size or scale_factor by using bilinear interpolate.

Inputs:

  • x (Tensor) - Tensor to be resized. Input tensor must be a 4-D tensor with shape: math:(batch, channels, height, width), with data type of float16 or float32.

  • size (Union[tuple[int], list[int]]): A tuple or list of 2 int elements ‚Äò(new_height, new_width)‚Äô, the new size of the tensor. One and only one of size and scale_factor can be set to None. Default: None.

  • scale_factor (int): The scale factor of new size of the tensor. The value should be positive integer. One and only one of size and scale_factor can be set to None. Default: None.

  • align_corners (bool): If true, rescale input by ‚Äò(new_height - 1) / (height - 1)‚Äô, which exactly aligns the 4 corners of images and resized images. If false, rescale by ‚Äònew_height / height‚Äô. Default: False.

Outputs:

Resized tensor. If size is set, the result is 4-D tensor with shape:math:(batch, channels, new_height, new_width) in float32. If scale is set, the result is 4-D tensor with shape:math:(batch, channels, scale_factor * height, scale_factor * width) in float32

Raises

  • TypeError ‚Äì If size is not one of tuple, list, None.

  • TypeError ‚Äì If scale_factor is neither int nor None.

  • TypeError ‚Äì If align_corners is not a bool.

  • TypeError ‚Äì If dtype of x is neither float16 nor float32.

  • ValueError ‚Äì If size and scale_factor are both None or not None.

  • ValueError ‚Äì If length of shape of x is not equal to 4.

  • ValueError ‚Äì If scale_factor is an int which is less than 1.

  • ValueError ‚Äì If size is a list or tuple whose length is not equal to 2.

Supported Platforms:

Ascend CPU
