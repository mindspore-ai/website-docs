mindspore.dataset.audio.DensityFunction

class mindspore.dataset.audio.DensityFunction[source]

Density Functions.

Possible enumeration values are: DensityFunction.TPDF, DensityFunction.RPDF, DensityFunction.GPDF.

  • DensityFunction.TPDF: means triangular probability density function.

  • DensityFunction.RPDF: means rectangular probability density function.

  • DensityFunction.GPDF: means gaussian probability density function.
