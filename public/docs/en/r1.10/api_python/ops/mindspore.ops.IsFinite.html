mindspore.ops.IsFinite

class mindspore.ops.IsFinite[source]

Determines which elements are finite for each position.

Refer to mindspore.ops.isfinite() for more detail.

Supported Platforms:

Ascend GPU CPU

Examples

>>> is_finite = ops.IsFinite()
>>> x = Tensor(np.array([np.log(-1), 1, np.log(0)]), mindspore.float32)
>>> output = is_finite(x)
>>> print(output)
[False  True False]
