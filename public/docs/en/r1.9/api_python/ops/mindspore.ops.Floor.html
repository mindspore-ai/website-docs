mindspore.ops.Floor

class mindspore.ops.Floor[source]

Rounds a tensor down to the closest integer element-wise.

Refer to mindspore.ops.floor() for more detail.

Supported Platforms:

Ascend GPU CPU

Examples

>>> x = Tensor(np.array([1.1, 2.5, -1.5]), mindspore.float32)
>>> floor = ops.Floor()
>>> output = floor(x)
>>> print(output)
[ 1.  2. -2.]
