.. py:method:: mindspore.dataset.Dataset.get_class_indexing()

    获取类别名称到类别索引的映射字典。

    该字典可用于查询某一类别索引对应的是哪一个类别名称。

    返回：
        Dict[str, int]，类别名称到类别索引的映射关系。
