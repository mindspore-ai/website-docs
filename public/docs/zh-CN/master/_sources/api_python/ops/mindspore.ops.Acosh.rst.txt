.. py:class:: mindspore.ops.Acosh

    逐元素计算输入Tensor的反双曲余弦。

    获取更多详情请查看： :func:`mindspore.ops.acosh` 。

    输入：
        - **x** (Tensor) - 输入Tensor。取值必须在[1, inf]范围内。

    输出：
        Tensor，shape和数据类型与 `x` 相同。
