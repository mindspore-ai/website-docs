mindspore.ops.BesselK1e
=======================

.. py:class:: mindspore.ops.BesselK1e

    逐元素计算输入数据的BesselK1e函数值。

    输入：
        - **x** (Tensor) - 输入Tensor。数据类型应为float16、float32或float64。

    输出：
        Tensor，shape和数据类型与 `x` 相同。

    异常：
        - **TypeError** - `x` 不是float16、float32或float64数据类型的Tensor。
