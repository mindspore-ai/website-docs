.. py:class:: mindspore.ops.Real

    返回输入Tensor的实数部分。如果输入是实数，则返回值不变。

    输入：
        - **input** (Tensor) - 要计算的输入Tensor。

    输出：
        Tensor，shape与输入 `input` 相同。

    异常：
        - **TypeError** - 如果 `input` 不是Tensor。
