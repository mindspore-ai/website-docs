.. py:class:: mindspore.ops.Trunc

    返回一个新的张量，该张量具有输入元素的截断整数值。

    更多参考详见 :func:`mindspore.ops.trunc`。

    输入：
        - **input_x** (Tensor) - 任意维度的Tensor。

    输出：
        Tensor，shape和数据类型与 `input_x` 相同。
