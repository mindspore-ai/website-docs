.. py:function:: mindspore.ops.is_complex(input)

    如果Tensor的数据类型是复数，则返回True，否则返回False。

    参数：
        - **input** (Tensor) - 输入Tensor。

    返回：
        Bool，返回Tensor的数据类型是否为复数。

    异常：
        - **TypeError** - `input` 不是Tensor。
