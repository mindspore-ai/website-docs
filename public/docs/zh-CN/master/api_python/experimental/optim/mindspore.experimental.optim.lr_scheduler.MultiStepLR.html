mindspore.experimental.optim.lr_scheduler.MultiStepLRпғҒ

class mindspore.experimental.optim.lr_scheduler.MultiStepLR(optimizer, milestones, gamma=0.1, last_epoch=- 1, verbose=False)[жәҗд»Јз Ғ]пғҒ

еҪ“epoch/stepиҫҫеҲ° milestones ж—¶пјҢе°ҶжҜҸдёӘеҸӮж•°з»„зҡ„еӯҰд№ зҺҮжҢүз…§д№ҳжі•еӣ еӯҗ gamma иҝӣиЎҢеҸҳеҢ–гҖӮжіЁж„ҸпјҢиҝҷз§ҚиЎ°еҮҸеҸҜиғҪдёҺеӨ–йғЁеҜ№дәҺеӯҰд№ зҺҮзҡ„ж”№еҸҳеҗҢж—¶еҸ‘з”ҹгҖӮ

иӯҰе‘Ҡ

иҝҷжҳҜдёҖдёӘе®һйӘҢжҖ§зҡ„еҠЁжҖҒеӯҰд№ зҺҮжҺҘеҸЈпјҢйңҖиҰҒе’Ң mindspore.experimental.optim дёӢзҡ„жҺҘеҸЈй…ҚеҗҲдҪҝз”ЁгҖӮ

еҸӮж•°пјҡ

  • optimizer (mindspore.experimental.optim.Optimizer) - дјҳеҢ–еҷЁе®һдҫӢгҖӮ

  • milestones (list) - иЎЁзӨәepoch/stepйҳҲеҖјзҡ„еҲ—иЎЁпјҢдёәйҖ’еўһеәҸеҲ—пјҢеҪ“epoch/stepж•°иҫҫеҲ°йҳҲеҖјж—¶е°ҶеӯҰд№ зҺҮд№ҳд»Ҙ gammaгҖӮ

  • gamma (floatпјҢеҸҜйҖү) - еӯҰд№ зҺҮзҡ„д№ҳжі•еӣ еӯҗгҖӮй»ҳи®ӨеҖјпјҡ 0.1гҖӮ

  • last_epoch (intпјҢеҸҜйҖү) - epoch/stepж•°гҖӮй»ҳи®ӨеҖјпјҡ-1гҖӮ

  • verbose (boolпјҢеҸҜйҖү) - жҳҜеҗҰжү“еҚ°еӯҰд№ зҺҮгҖӮй»ҳи®ӨеҖјпјҡ FalseгҖӮ
