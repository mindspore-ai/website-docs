mindspore.nn.HingeEmbeddingLossпғҒ

class mindspore.nn.HingeEmbeddingLoss(margin=1.0, reduction='mean')[жәҗд»Јз Ғ]пғҒ

Hinge Embedding жҚҹеӨұеҮҪж•°гҖӮж №жҚ®иҫ“е…Ҙ logits е’Ң labels пјҲеҸӘеҢ…еҗ«1жҲ–-1пјүи®Ўз®—Hinge EmbeddingжҚҹеӨұеҖјгҖӮйҖҡеёёиў«з”ЁжқҘиЎЎйҮҸдёӨдёӘиҫ“е…Ҙд№Ӣй—ҙзҡ„зӣёдјјеәҰгҖӮ

mini-batchдёӯзҡ„з¬¬ \(n\) дёӘж ·дҫӢзҡ„жҚҹеӨұеҮҪж•°дёәпјҡ

\[\begin{split}l_n = \begin{cases} x_n, & \text{if}\; y_n = 1,\\ \max \{0, \Delta - x_n\}, & \text{if}\; y_n = -1, \end{cases}\end{split}\]

жҖ»жҚҹеӨұеҖјдёәпјҡ

\[\begin{split}\ell(x, y) = \begin{cases} \operatorname{mean}(L), & \text{if reduction} = \text{'mean';}\\ \operatorname{sum}(L), & \text{if reduction} = \text{'sum'.} \end{cases}\end{split}\]

е…¶дёӯ \(L = \{l_1,\dots,l_N\}^\top\)гҖӮ

еҸӮж•°пјҡ

  • margin (float, int) - Hinge Embedding Lossе…¬ејҸе®ҡд№үзҡ„йҳҲеҖј \(margin\)гҖӮе…¬ејҸдёӯиЎЁзӨәдёә \(\Delta\)гҖӮй»ҳи®ӨеҖјпјҡ 1.0гҖӮ

  • reduction (strпјҢеҸҜйҖү) - жҢҮе®ҡеә”з”ЁдәҺиҫ“еҮәз»“жһңзҡ„и§„зәҰи®Ўз®—ж–№ејҸпјҢеҸҜйҖү 'none' гҖҒ 'mean' гҖҒ 'sum' пјҢй»ҳи®ӨеҖјпјҡ 'mean' гҖӮ

    • "none"пјҡдёҚеә”з”Ёи§„зәҰж–№жі•гҖӮ

    • "mean"пјҡи®Ўз®—иҫ“еҮәе…ғзҙ зҡ„е№іеқҮеҖјгҖӮ

    • "sum"пјҡи®Ўз®—иҫ“еҮәе…ғзҙ зҡ„жҖ»е’ҢгҖӮ

иҫ“е…Ҙпјҡ

  • logits (Tensor) - йў„жөӢеҖјпјҢе…¬ејҸдёӯиЎЁзӨәдёә \(x\)пјҢshapeдёә \((*)\)гҖӮ \(*\) д»ЈиЎЁзқҖд»»ж„Ҹж•°йҮҸзҡ„з»ҙеәҰгҖӮ

  • labels (Tensor) - ж ҮзӯҫеҖјпјҢе…¬ејҸдёӯиЎЁзӨәдёә \(y\)пјҢе’Ң logits е…·жңүзӣёеҗҢshapeпјҢеҢ…еҗ«1жҲ–-1гҖӮ

иҝ”еӣһпјҡ

TensorжҲ–Tensor scalarпјҢж №жҚ® \(reduction\) и®Ўз®—зҡ„lossгҖӮ

ејӮеёёпјҡ

  • TypeError - logits дёҚжҳҜTensorгҖӮ

  • TypeError - labels дёҚжҳҜTensorгҖӮ

  • TypeError - margin дёҚжҳҜfloatжҲ–intгҖӮ

  • ValueError - labels е’Ң logits shapeдёҚдёҖиҮҙдё”дёҚиғҪе№ҝж’ӯгҖӮ

  • ValueError - reduction дёҚжҳҜвҖқnoneвҖқгҖҒвҖқmeanвҖқжҲ–иҖ…вҖқsumвҖқгҖӮ

ж”ҜжҢҒе№іеҸ°пјҡ

ж ·дҫӢпјҡ

>>> import mindspore as ms
>>> import mindspore.nn as nn
>>> import numpy as np
>>> arr1 = np.array([0.9, -1.2, 2, 0.8, 3.9, 2, 1, 0, -1]).reshape((3, 3))
>>> arr2 = np.array([1, 1, -1, 1, -1, 1, -1, 1, 1]).reshape((3, 3))
>>> logits = ms.Tensor(arr1, ms.float32)
>>> labels = ms.Tensor(arr2, ms.float32)
>>> loss = nn.HingeEmbeddingLoss(reduction='mean')
>>> output = loss(logits, labels)
>>> print(output)
