mindspore.nn.Identity

class mindspore.nn.Identity[源代码]

网络占位符，返回与输入完全一致。

输入：

  • x (Any) - Identity的输入。

输出：

与 x 完全相同。

支持平台：

Ascend GPU CPU

样例：

>>> import mindspore
>>> from mindspore import Tensor, nn
>>> import numpy as np
>>> x = Tensor(np.array([1, 2, 3, 4]), mindspore.int64)
>>> net = nn.Identity()
>>> output = net(x)
>>> print(output)
[1 2 3 4]
