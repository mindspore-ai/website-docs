mindspore.nn.MultiheadAttentionпғҒ

class mindspore.nn.MultiheadAttention(embed_dim, num_heads, dropout=0.0, has_bias=True, add_bias_kv=False, add_zero_attn=False, kdim=None, vdim=None, batch_first=False, dtype=mstype.float32)[жәҗд»Јз Ғ]пғҒ

и®әж–Ү Attention Is All You Need дёӯжүҖиҝ°зҡ„еӨҡеӨҙжіЁж„ҸеҠӣзҡ„е®һзҺ°гҖӮз»ҷе®ҡqueryеҗ‘йҮҸпјҢkeyеҗ‘йҮҸе’ҢvalueпјҢжіЁж„ҸеҠӣи®Ўз®—жөҒзЁӢеҰӮдёӢпјҡ

\[MultiHeadAttention(query, key, vector) = Concat(head_1, \dots, head_h)W^O\]

е…¶дёӯпјҢ \(head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)\) гҖӮжіЁж„Ҹпјҡиҫ“еҮәеұӮзҡ„жҠ•еҪұи®Ўз®—дёӯеёҰжңүеҒҸзҪ®еҸӮж•°гҖӮ

еҰӮжһңqueryгҖҒkeyе’ҢvalueзӣёеҗҢпјҢеҲҷдёҠиҝ°еҚідёәиҮӘжіЁж„ҸеҠӣжңәеҲ¶зҡ„и®Ўз®—иҝҮзЁӢгҖӮ

еҸӮж•°пјҡ

  • embed_dim (int) - жЁЎеһӢзҡ„жҖ»з»ҙж•°гҖӮ

  • num_heads (int) - е№¶иЎҢжіЁж„ҸеҠӣеӨҙзҡ„ж•°йҮҸгҖӮnum_heads йңҖиҰҒиғҪеӨҹиў« embed_dim ж•ҙйҷӨпјҲжҜҸдёӘеӨҙзҡ„з»ҙж•°дёә embed_dim // num_headsпјүгҖӮ

  • dropout (float) - еә”з”ЁеҲ°иҫ“е…Ҙ attn_output_weights дёҠзҡ„йҡҸжңәдёўејғжҜ”дҫӢ. й»ҳи®ӨеҖјпјҡ 0.0гҖӮ

  • has_bias (bool) - жҳҜеҗҰз»ҷиҫ“е…ҘгҖҒиҫ“еҮәжҠ•е°„еұӮж·»еҠ еҒҸзҪ®гҖӮй»ҳи®ӨеҖјпјҡ TrueгҖӮ

  • add_bias_kv (bool) - жҳҜеҗҰз»ҷkeyгҖҒvalueеәҸеҲ—зҡ„0з»ҙж·»еҠ еҒҸзҪ®гҖӮй»ҳи®ӨеҖјпјҡ FalseгҖӮ

  • add_zero_attn (bool) - жҳҜеҗҰз»ҷkeyгҖҒvalueеәҸеҲ—зҡ„дёҖз»ҙж·»еҠ 0гҖӮй»ҳи®ӨеҖјпјҡ FalseгҖӮ

  • kdim (int) - keyзҡ„жҖ»зү№еҫҒж•°гҖӮй»ҳи®ӨеҖјпјҡ None пјҲеҚі kdim=embed_dimпјүгҖӮ

  • vdim (int) - valueзҡ„жҖ»зү№еҫҒж•°гҖӮй»ҳи®ӨеҖјпјҡNone пјҲеҚі vdim=embed_dimпјүгҖӮ

  • batch_first (bool) - еҰӮжһңдёә TrueпјҢеҲҷиҫ“е…Ҙиҫ“еҮәTensorзҡ„shapeдёә \((batch, seq, feature)\) пјҢеҗҰеҲҷshapeдёә \((seq, batch, feature)\) гҖӮ й»ҳи®ӨеҖјпјҡ False гҖӮ

  • dtype (mindspore.dtype) - Parameterзҡ„ж•°жҚ®зұ»еһӢгҖӮй»ҳи®ӨеҖјпјҡ mstype.float32 гҖӮ

иҫ“е…Ҙпјҡ

  • query (Tensor) - Queryзҹ©йҳөгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((L, E_q)\) гҖӮеҪ“иҫ“е…ҘBatchж•°жҚ®пјҢеҸӮж•° batch_first=False ж—¶пјҢShapeдёә \((L, N, E_q)\) пјҢ еҪ“ batch_first=True ж—¶пјҢShapeдёә \((N, L, E_q)\)гҖӮе…¶дёӯпјҢ \(L\) дёәзӣ®ж ҮеәҸеҲ—зҡ„й•ҝеәҰпјҢ \(N\) дёәbatch sizeпјҢ\(E_q\) дёәQueryзҹ©йҳөзҡ„з»ҙж•° embed_dimгҖӮ ж•°жҚ®зұ»еһӢпјҡfloat16гҖҒfloat32жҲ–иҖ…float64гҖӮжіЁж„ҸеҠӣжңәеҲ¶йҖҡиҝҮQueryдёҺKey-Valueиҝҗз®—д»Ҙз”ҹжҲҗжңҖз»Ҳиҫ“еҮәгҖӮиҜҰжғ…иҜ·и§ҒпјҡвҖқAttention Is All You NeedвҖқгҖӮ

  • key (Tensor) - Keyзҹ©йҳөгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((S, E_k)\) гҖӮеҪ“иҫ“е…ҘBatchж•°жҚ®пјҢеҸӮж•° batch_first=False ж—¶пјҢShapeдёә \((S, N, E_k)\) пјҢ еҪ“ batch_first=True ж—¶пјҢShapeдёә \((N, S, E_k)\)гҖӮе…¶дёӯпјҢ \(S\) дёәжәҗеәҸеҲ—зҡ„й•ҝеәҰпјҢ \(N\) дёәbatch sizeпјҢ\(E_k\) дёәKeyзҹ©йҳөзҡ„з»ҙж•° kdimгҖӮж•°жҚ®зұ»еһӢпјҡfloat16гҖҒfloat32жҲ–иҖ…float64гҖӮиҜҰжғ…иҜ·и§ҒпјҡвҖқAttention Is All You NeedвҖқгҖӮ

  • value (Tensor) - Valueзҹ©йҳөгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((S, E_v)\) гҖӮеҪ“иҫ“е…ҘBatchж•°жҚ®пјҢеҸӮж•° batch_first=False ж—¶пјҢShapeдёә \((S, N, E_v)\) пјҢ еҪ“ batch_first=True ж—¶пјҢShapeдёә \((N, S, E_v)\)гҖӮе…¶дёӯпјҢ \(S\) дёәжәҗеәҸеҲ—зҡ„й•ҝеәҰпјҢ \(N\) дёәbatch sizeпјҢ\(E_v\) дёәKeyзҹ©йҳөзҡ„з»ҙж•° vdimгҖӮж•°жҚ®зұ»еһӢпјҡfloat16гҖҒfloat32жҲ–иҖ…float64гҖӮиҜҰжғ…иҜ·и§ҒпјҡвҖқAttention Is All You NeedвҖқгҖӮ

  • key_padding_mask (Tensor, optional) - еҰӮжһңжҢҮе®ҡжӯӨеҖјпјҢеҲҷиЎЁзӨәShapeдёә \((N, S)\)гҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((S)\) гҖӮ еҰӮжһңиҫ“е…ҘTensorдёәBoolзұ»еһӢпјҢеҲҷ key дёӯеҜ№еә”дёә True зҡ„дҪҚзҪ®е°ҶеңЁAttentionи®Ўз®—ж—¶иў«еҝҪз•ҘгҖӮеҰӮжһңиҫ“е…ҘTensorдёәFloatзұ»еһӢпјҢеҲҷе°ҶзӣҙжҺҘдёҺ key зӣёеҠ гҖӮFloatж”ҜжҢҒж•°жҚ®зұ»еһӢпјҡfloat16гҖҒfloat32жҲ–иҖ…float64гҖӮй»ҳи®ӨеҖјпјҡNoneгҖӮ

  • need_weights (bool) - жҳҜеҗҰйңҖиҰҒиҝ”еӣһ attn_output_weightsпјҢеҰӮжһңдёә TrueпјҢеҲҷиҫ“еҮәеҢ…еҗ« attn_output_weightsгҖӮй»ҳи®ӨеҖјпјҡTrueгҖӮ

  • attn_mask (Tensor, optional) - еҰӮжһңжҢҮе®ҡжӯӨеҖјпјҢеҲҷиЎЁзӨәShapeдёә \((L, S)\) жҲ– \((N\cdot\text{num_heads}, L, S)\) зҡ„жҺ©з Ғе°Ҷиў«з”ЁдәҺAttentionи®Ўз®—гҖӮе…¶дёӯ \(N\) дёәbatch sizeпјҢ \(L\) дёәзӣ®ж ҮеәҸеҲ—й•ҝеәҰпјҢ\(S\) дёәжәҗеәҸеҲ—й•ҝеәҰгҖӮеҰӮжһңиҫ“е…Ҙдёә2з»ҙзҹ©йҳөпјҢеҲҷе°ҶиҮӘеҠЁжІҝbatchз»ҙе№ҝж’ӯиҮі3з»ҙзҹ©йҳөгҖӮиӢҘдёә3з»ҙзҹ©йҳөпјҢеҲҷе…Ғи®ёжІҝbatchз»ҙдҪҝз”ЁдёҚеҗҢзҡ„жҺ©з ҒгҖӮеҰӮжһңиҫ“е…ҘTensorдёәBoolзұ»еһӢпјҢеҲҷеҖјдёә True еҜ№еә”дҪҚзҪ®е…Ғи®ёиў«жіЁж„ҸеҠӣи®Ўз®—гҖӮеҰӮжһңиҫ“е…ҘTensorдёәFloatзұ»еһӢпјҢеҲҷе°ҶзӣҙжҺҘдёҺжіЁж„ҸеҠӣжқғйҮҚзӣёеҠ гҖӮFloatж”ҜжҢҒж•°жҚ®зұ»еһӢпјҡfloat16гҖҒfloat32жҲ–иҖ…float64гҖӮй»ҳи®ӨеҖјпјҡNoneгҖӮ

  • average_attn_weights (bool) - еҰӮжһңдёә TrueпјҢ еҲҷиҝ”еӣһеҖј attn_weights дёәжіЁж„ҸеҠӣеӨҙзҡ„е№іеқҮеҖјгҖӮеҰӮжһңдёә FalseпјҢеҲҷ attn_weights еҲҶеҲ«иҝ”еӣһжҜҸдёӘжіЁж„ҸеҠӣеӨҙзҡ„еҖјгҖӮ жң¬еҸӮж•°д»…еңЁ need_weights=True ж—¶з”ҹж•ҲгҖӮй»ҳи®ӨеҖјпјҡ True гҖӮ

иҫ“еҮәпјҡ

TupleпјҢиЎЁзӨәдёҖдёӘеҢ…еҗ«(attn_output, attn_output_weights)зҡ„е…ғз»„гҖӮ

  • attn_output - жіЁж„ҸеҠӣжңәеҲ¶зҡ„иҫ“еҮәгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((L, E)\) гҖӮеҪ“иҫ“е…ҘBatchж•°жҚ®пјҢ еҸӮж•° batch_first=False ж—¶пјҢShapeдёә \((L, N, E)\) пјҢ еҪ“ batch_first=True ж—¶пјҢShapeдёә \((N, L, E)\)гҖӮе…¶дёӯпјҢ \(L\) дёәзӣ®ж ҮеәҸеҲ—зҡ„й•ҝеәҰпјҢ \(N\) дёәbatch sizeпјҢ \(E\) дёәжЁЎеһӢзҡ„жҖ»з»ҙж•° embed_dimгҖӮ

  • attn_output_weights - д»…еҪ“ need_weights=True ж—¶иҝ”еӣһгҖӮеҰӮжһң average_attn_weights=TrueпјҢеҲҷиҝ”еӣһеҖј attn_weights дёәжіЁж„ҸеҠӣеӨҙзҡ„е№іеқҮеҖјгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢ Shapeдёәпјҡ \((L, S)\) пјҢеҪ“иҫ“е…ҘBatchж•°жҚ®ж—¶пјҢShapeдёә \((N, L, S)\)гҖӮе…¶дёӯ \(N\) дёәbatch sizeпјҢ \(L\) дёәзӣ®ж ҮеәҸеҲ—зҡ„й•ҝеәҰпјҢ\(S\) дёәжәҗеәҸеҲ—й•ҝеәҰгҖӮ еҰӮжһң average_attn_weights=False пјҢеҲҶеҲ«иҝ”еӣһжҜҸдёӘжіЁж„ҸеҠӣеӨҙзҡ„еҖјгҖӮеҪ“иҫ“е…ҘйқһBatchж•°жҚ®ж—¶пјҢShapeдёәпјҡ \((\text{num_heads}, L, S)\) пјҢеҪ“иҫ“е…ҘBatchж•°жҚ®ж—¶пјҢShapeдёә \((N, \text{num_heads}, L, S)\)гҖӮ

ејӮеёёпјҡ

  • ValueError - еҰӮжһң embed_dim дёҚиғҪиў« num_heads ж•ҙйҷӨгҖӮ

  • TypeError - еҰӮжһң key_padding_mask дёҚжҳҜboolжҲ–floatзұ»еһӢгҖӮ

ж”ҜжҢҒе№іеҸ°пјҡ

Ascend GPU CPU

ж ·дҫӢпјҡ

>>> import mindspore as ms
>>> import numpy as np
>>> embed_dim, num_heads = 128, 8
>>> seq_length, batch_size = 10, 8
>>> query = ms.Tensor(np.random.randn(seq_length, batch_size, embed_dim), ms.float32)
>>> key = ms.Tensor(np.random.randn(seq_length, batch_size, embed_dim), ms.float32)
>>> value = ms.Tensor(np.random.randn(seq_length, batch_size, embed_dim), ms.float32)
>>> multihead_attn = ms.nn.MultiheadAttention(embed_dim, num_heads)
>>> attn_output, attn_output_weights = multihead_attn(query, key, value)
>>> print(attn_output.shape)
(10, 8, 128)
