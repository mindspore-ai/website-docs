mindspore.ops.IOUпғҒ

class mindspore.ops.IOU(mode='iou')[жәҗд»Јз Ғ]пғҒ

и®Ўз®—зҹ©еҪўзҡ„IOUпјҢеҚізңҹе®һеҢәеҹҹе’Ңйў„жөӢеҢәеҹҹзҡ„дәӨе№¶жҜ”гҖӮ

ж №жҚ®зңҹе®һеҢәеҹҹе’Ңйў„жөӢеҢәеҹҹи®Ўз®—IOU(intersection over union)жҲ–IOF(intersection over foreground)гҖӮ

жӣҙеӨҡеҸӮиҖғиҜҰи§Ғ mindspore.ops.iou()гҖӮ

еҸӮж•°пјҡ

  • mode (string) - жҢҮе®ҡи®Ўз®—ж–№жі•пјҢзҺ°ж”ҜжҢҒ 'iou' (intersection over union)жҲ– 'iof' (intersection over foreground)жЁЎејҸгҖӮй»ҳи®ӨеҖјпјҡ 'iou' гҖӮ

иҫ“е…Ҙпјҡ

  • anchor_boxes (Tensor) - йў„жөӢеҢәеҹҹпјҢshapeдёә \((N, 4)\) зҡ„TensorгҖӮвҖқNвҖқиЎЁзӨәйў„жөӢеҢәеҹҹзҡ„ж•°йҮҸпјҢвҖқ4вҖқиЎЁзӨәвҖқx0вҖқгҖҒвҖқy0вҖқгҖҒвҖқx1вҖқе’ҢвҖқy1вҖқгҖӮж•°жҚ®зұ»еһӢдёәfloat16жҲ–float32гҖӮ

  • gt_boxes (Tensor) - зңҹе®һеҢәеҹҹпјҢshapeдёә \((M, 4)\) зҡ„TensorгҖӮвҖқMвҖқиЎЁзӨәең°йқўзңҹе®һеҢәеҹҹзҡ„ж•°йҮҸпјҢвҖқ4вҖқиЎЁзӨәвҖқx0вҖқгҖҒвҖқy0вҖқгҖҒвҖқx1вҖқе’ҢвҖқy1вҖқгҖӮж•°жҚ®зұ»еһӢдёәfloat16жҲ–float32гҖӮ

иҫ“еҮәпјҡ

IOUеҖјзҡ„TensorпјҢshapeдёә \((M, N)\) зҡ„TensorпјҢж•°жҚ®зұ»еһӢдёҺ anchor_boxes зҡ„зӣёеҗҢгҖӮ

ж”ҜжҢҒе№іеҸ°пјҡ

Ascend GPU CPU

ж ·дҫӢпјҡ

>>> import mindspore
>>> import numpy as np
>>> from mindspore import Tensor, ops
>>> iou = ops.IOU(mode='iou')
>>> anchor_boxes = Tensor(np.random.randint(1.0, 5.0, [3, 4]), mindspore.float16)
>>> gt_boxes = Tensor(np.random.randint(1.0, 5.0, [3, 4]), mindspore.float16)
>>> output = iou(anchor_boxes, gt_boxes)
>>> print(output.shape)
(3, 3)
