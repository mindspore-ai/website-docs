mindspore.ops.ReLU6

class mindspore.ops.ReLU6[源代码]

计算输入Tensor的ReLU（矫正线性单元），其上限为6。

更多参考详见 mindspore.ops.relu6()。

输入：

  • input_x (Tensor) - 输入Tensor。shape为 \((N, *)\) ，其中 \(*\) 表示任意的附加维度数。数据类型必须为float16、float32。

输出：

Tensor，数据类型和shape与 input_x 相同。

支持平台：

Ascend GPU CPU

样例：

>>> import mindspore
>>> import numpy as np
>>> from mindspore import Tensor, ops
>>> input_x = Tensor(np.array([[-1.0, 4.0, -8.0], [2.0, -5.0, 9.0]]), mindspore.float32)
>>> relu6 = ops.ReLU6()
>>> result = relu6(input_x)
>>> print(result)
[[0. 4. 0.]
 [2. 0. 6.]]
