е·®ејӮеҜ№жҜ”пғҒ

PyTorchпјҡи®Ўз®—дёҖз»ҙиҪ¬зҪ®еҚ·з§ҜпјҢеҸҜд»Ҙи§ҶдёәConv1dеҜ№иҫ“е…ҘжұӮжўҜеәҰпјҢд№ҹз§°дёәеҸҚеҚ·з§ҜпјҲе®һйҷ…дёҚжҳҜзңҹжӯЈзҡ„еҸҚеҚ·з§ҜпјүгҖӮиҫ“е…Ҙзҡ„shapeйҖҡеёёжҳҜ\((N,C_{in}, L_{in})\)пјҢе…¶дёӯ\(N\)жҳҜbatch sizeпјҢ\(C\)жҳҜз©әй—ҙз»ҙеәҰпјҢ\(L\)жҳҜеәҸеҲ—зҡ„й•ҝеәҰгҖӮиҫ“еҮәзҡ„shapeдёә\((N,C_{out},L_{out})\)пјҢе…¶дёӯ\(L_{out}=(L_{in}вҲ’1)Г—strideвҲ’2Г—padding+dilationГ—(kernel\_sizeвҲ’1)+output\_padding+1\)

MindSporeпјҡMindSporeжӯӨAPIе®һзҺ°еҠҹиғҪдёҺPyTorchеҹәжң¬дёҖиҮҙпјҢж–°еўһдәҶеЎ«е……жЁЎејҸеҸӮж•°вҖқpad_modeвҖқпјҢеҪ“вҖқpad_modeвҖқ = вҖңpadвҖқж—¶дёҺPyTorchй»ҳи®Өж–№ејҸзӣёеҗҢпјҢеҲ©з”Ёweight_init е’Ңbias_init еҸӮж•°еҸҜд»Ҙй…ҚзҪ®еҲқе§ӢеҢ–ж–№ејҸгҖӮ

жқғйҮҚеҲқе§ӢеҢ–е·®ејӮпғҒ

mindspore.nn.Conv1dTransposeзҡ„ weight_init жҳҜ None ж—¶пјҢжқғйҮҚдҪҝз”ЁHeUniformеҲқе§ӢеҢ–гҖӮжӯӨж—¶е’ҢPyTorchжқғйҮҚеҲқе§ӢеҢ–ж–№ејҸдёҖиҮҙгҖӮ

mindspore.nn.Conv1dTransposeзҡ„ bias_init жҳҜ None ж—¶пјҢеҒҸе·®дҪҝз”ЁUniformеҲқе§ӢеҢ–гҖӮжӯӨж—¶е’ҢPyTorchеҒҸе·®еҲқе§ӢеҢ–ж–№ејҸдёҖиҮҙгҖӮ

д»Јз ҒзӨәдҫӢ1пғҒ

дёӨAPIйғҪжҳҜе®һзҺ°дёҖз»ҙиҪ¬зҪ®еҚ·з§Ҝиҝҗз®—пјҢдҪҝз”Ёж—¶йңҖе…ҲиҝӣиЎҢе®һдҫӢеҢ–гҖӮеҪ“PyTorchе°Ҷoutput_paddingи®ҫдёә0пјҢMindSporeе°Ҷpad_modeи®ҫдёәвҖқpadвҖқж—¶пјҢиҫ“еҮәе®ҪеәҰ\(L_{out}=(L_{in}вҲ’1)Г—strideвҲ’2Г—padding+dilationГ—(kernel\_sizeвҲ’1)+1\)гҖӮPyTorchдёӯеҲ©з”Ёnet.weight.data = torch.ones()зҡ„ж–№ејҸе°ҶжқғйҮҚеҲқе§ӢеҢ–дёә1пјҢMindSporeзӣҙжҺҘи®ҫзҪ®еҸӮж•°weight_init = вҖңonesвҖқгҖӮ

# PyTorch
import torch
from torch import tensor
import torch.nn as nn
import numpy as np

k = 4
x_ = np.ones([1, 3, 50])
x = tensor(x_, dtype=torch.float32)
net = nn.ConvTranspose1d(3, 64, kernel_size=k, stride=1, padding=0, output_padding=0, bias=False)
net.weight.data = torch.ones(3, 64, k)
output = net(x).detach().numpy()
print(output.shape)
# (1, 64, 53)


# MindSpore
import mindspore as ms
import mindspore.nn as nn
import numpy as np

k = 4
x_ = np.ones([1, 3, 50])
x = ms.Tensor(x_, ms.float32)
net = nn.Conv1dTranspose(3, 64, kernel_size=k, weight_init='ones', pad_mode='pad')
output = net(x)
print(output.shape)
# (1, 64, 53)

д»Јз ҒзӨәдҫӢ2пғҒ

дёәдҪҝиҫ“еҮәзҡ„е®ҪеәҰдёҺиҫ“е…Ҙж•ҙйҷӨstrideеҗҺзҡ„еҖјзӣёеҗҢпјҢPyTorchдёӯи®ҫзҪ®output_padding = stride - 1пјҢpaddingи®ҫзҪ®дёә(kernel_size - 1)/2гҖӮMindSporeеҲҷи®ҫзҪ®pad_mode = вҖңsameвҖқпјҢеҗҢж—¶padding = 0гҖӮ

# PyTorch
import torch
from torch import tensor
import torch.nn as nn
import numpy as np

k = 5
s = 3
x_ = np.ones([1, 3, 50])
x = tensor(x_, dtype=torch.float32)
net = nn.ConvTranspose1d(3, 64, kernel_size=k, stride=s, padding=(k-1)//2, output_padding=s-1, bias=False)
net.weight.data = torch.ones(3, 64, k)
output = net(x).detach().numpy()
print(output.shape)
# (1, 64, 150)


# MindSpore
import mindspore as ms
import mindspore.nn as nn
import numpy as np

k = 5
s = 3
x_ = np.ones([1, 3, 50])
x = ms.Tensor(x_, ms.float32)
net = nn.Conv1dTranspose(3, 64, kernel_size=k, stride=s, weight_init='ones', pad_mode='same', padding=0)
output = net(x)
print(output.shape)
# (1, 64, 150)

д»Јз ҒзӨәдҫӢ3пғҒ

иӢҘдёҚеңЁеҺҹжңүеӣҫеғҸдёҠеҒҡд»»дҪ•еЎ«е……пјҢеңЁstride>1зҡ„жғ…еҶөдёӢеҸҜиғҪиҲҚејғдёҖйғЁеҲҶж•°жҚ®пјҢиҫ“еҮәе®ҪеәҰ\(L_{out}=(L_{in}вҲ’1)Г—stride+dilationГ—(kernel\_sizeвҲ’1)+1\)гҖӮеңЁPyTorchдёӯе°Ҷpaddingе’Ңoutput_paddingи®ҫдёә0пјҢMindSporeдёӯи®ҫзҪ®pad_mode = вҖңvalidвҖқгҖӮ

# PyTorch
import torch
from torch import tensor
import torch.nn as nn
import numpy as np

k = 5
s = 3
x_ = np.ones([1, 3, 50])
x = tensor(x_, dtype=torch.float32)
net = nn.ConvTranspose1d(3, 64, kernel_size=k, stride=s, padding=0, output_padding=0, bias=False)
net.weight.data = torch.ones(3, 64, k)
output = net(x).detach().numpy()
print(output.shape)
# (1, 64, 152)


# MindSpore
import mindspore as ms
import mindspore.nn as nn
import numpy as np

k = 5
s = 3
x_ = np.ones([1, 3, 50])
x = ms.Tensor(x_, ms.float32)
