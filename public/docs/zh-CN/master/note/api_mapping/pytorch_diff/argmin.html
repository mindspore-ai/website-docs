е·®ејӮеҜ№жҜ”пғҒ

PyTorchпјҡиҝ”еӣһTensorеұ•е№іжҲ–жІҝзқҖз»ҷе®ҡзҡ„з»ҙеәҰжңҖе°ҸеҖјзҡ„зҙўеј•пјҢиҝ”еӣһеҖјзұ»еһӢдёәtorch.int64гҖӮеҰӮжһңжңүеӨҡдёӘжңҖе°ҸеҖјпјҢеҲҷиҝ”еӣһз¬¬дёҖдёӘжңҖе°ҸеҖјзҡ„зҙўеј•гҖӮ

MindSporeпјҡMindSporeжӯӨAPIе®һзҺ°еҠҹиғҪдёҺPyTorchеҹәжң¬дёҖиҮҙпјҢиҝ”еӣһеҖјзұ»еһӢдёәint32.

дёәдҝқиҜҒдәҢиҖ…иҫ“еҮәзұ»еһӢжҳҜдёҖиҮҙзҡ„пјҢйңҖдҪҝз”Ёmindspore.ops.Castз®—еӯҗе°ҶMindSporeзҡ„и®Ўз®—з»“жһңиҪ¬жҚўжҲҗmindspore.int64пјҢд»ҘдёӢжҜҸдёӘзӨәдҫӢеқҮдјҡжңүжӯӨжӯҘзұ»еһӢиҪ¬жҚўгҖӮ

д»Јз ҒзӨәдҫӢ1пғҒ

еҜ№дәҺйӣ¶з»ҙзҡ„TensorпјҢPyTorchж”ҜжҢҒdimеҸӮж•°дёәNone/-1/0еҸҠkeepdimеҸӮж•°дёәTrue/Falseзҡ„д»»ж„Ҹз»„еҗҲпјҢдё”и®Ўз®—з»“жһңйғҪжҳҜдёҖиҮҙзҡ„пјҢйғҪжҳҜдёҖдёӘйӣ¶з»ҙTensorгҖӮMindSpore 1.8.1зүҲжң¬жҡӮж—¶дёҚж”ҜжҢҒеӨ„зҗҶйӣ¶з»ҙTensorпјҢйңҖиҰҒе…ҲдҪҝз”Ёmindspore.ops.ExpandDimsе°ҶTensorжү©е……дёәдёҖз»ҙпјҢз„¶еҗҺеҶҚжҢүз…§mindspore.ops.argminз®—еӯҗзҡ„й»ҳи®ӨеҸӮж•°и®Ўз®—гҖӮ

# PyTorch
import torch
import numpy as np

x = np.arange(1).reshape(()).astype(np.float32)
torch_argmin = torch.argmin
torch_output = torch_argmin(torch.tensor(x))
torch_out_np = torch_output.numpy()
print(torch_out_np)
# 0

# MindSpore
import numpy as np
import mindspore
from mindspore import Tensor

x = np.arange(1).reshape(()).astype(np.float32)
ms_argmin = mindspore.ops.argmin
ms_expanddims = mindspore.ops.ExpandDims()
ms_cast = mindspore.ops.Cast()
ms_tensor = Tensor(x)

if not ms_tensor.shape:
    ms_tensor_tmp = ms_expanddims(ms_tensor, 0)
    ms_output = ms_argmin(ms_tensor_tmp)

ms_output = ms_cast(ms_output, mindspore.int64)
ms_out_np = ms_output.asnumpy()
print(ms_out_np)
# 0

д»Јз ҒзӨәдҫӢ2пғҒ

PyTorchзҡ„argminз®—еӯҗеңЁдёҚжҳҫејҸз»ҷеҮәdimеҸӮж•°ж—¶пјҢи®Ўз®—з»“жһңжҳҜе°ҶеҺҹж•°з»„flattenеҗҺпјҢдҪңдёәдёҖз»ҙеј йҮҸеҒҡargminж“ҚдҪңзҡ„з»“жһңпјҢиҖҢMindSporeд»…ж”ҜжҢҒеҜ№еҚ•дёӘз»ҙеәҰиҝӣиЎҢи®Ўз®—гҖӮеӣ жӯӨпјҢдёәдәҶеҫ—еҲ°зӣёеҗҢзҡ„и®Ўз®—з»“жһңпјҢеңЁи®Ўз®—еүҚпјҢе°Ҷmindspore.ops.argminз®—еӯҗдј е…Ҙflattenзҡ„TensorеҚіеҸҜгҖӮ

# PyTorch
import torch
import numpy as np

x = np.arange(2*3*4).reshape(2, 3, 4).astype(np.float32)
torch_argmin = torch.argmin
torch_output = torch_argmin(torch.tensor(x))
torch_out_np = torch_output.numpy()
print(torch_out_np)
# 0

# MindSpore
import numpy as np
import mindspore
from mindspore import Tensor

dim = None
x = np.arange(2*3*4).reshape(2,3,4).astype(np.float32)
ms_argmin = mindspore.ops.argmin
ms_expanddims = mindspore.ops.ExpandDims()
ms_cast = mindspore.ops.Cast()
ms_tensor = Tensor(x)

ms_output = ms_argmin(ms_tensor, axis=dim) if dim is not None else ms_argmin(
    ms_tensor.flatten())

ms_output = ms_cast(ms_output, mindspore.int64)
ms_out_np = ms_output.asnumpy()
print(ms_out_np)
# 0

д»Јз ҒзӨәдҫӢ3пғҒ

PyTorchз®—еӯҗжңүдёҖдёӘkeepdimеҸӮж•°пјҢеҪ“и®ҫзҪ®дёәTrueж—¶пјҢдҪңз”Ёдёәпјҡе°ҶиҝӣиЎҢиҒҡеҗҲзҡ„з»ҙеәҰдҝқз•ҷпјҢе№¶и®ҫе®ҡдёә1гҖӮMindSporeзҡ„keepdimsеҸӮж•°дёҺе…¶еҠҹиғҪдёҖиҮҙгҖӮдёәдәҶе®һзҺ°зӣёеҗҢзҡ„з»“жһңпјҢеңЁи®Ўз®—е®ҢжҲҗеҗҺпјҢдҪҝз”Ёmindspore.ops.ExpandDimsз®—еӯҗжү©е……з»ҙеәҰеҚіеҸҜгҖӮ

# PyTorch
import torch
import numpy as np

dim = 1
keepdims = True
x = np.arange(2*4).reshape(2, 4).astype(np.float32)
torch_argmin = torch.argmin
torch_output = torch_argmin(torch.tensor(x), dim=dim, keepdims=keepdims)
torch_out_np = torch_output.numpy()
print(torch_out_np)
