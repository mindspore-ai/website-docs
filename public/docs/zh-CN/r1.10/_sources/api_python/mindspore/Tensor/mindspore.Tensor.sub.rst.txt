mindspore.Tensor.sub
====================

    更多细节参考 :func:`mindspore.ops.sub`。

    参数：
        - **y** (Union[Tensor, number.Number, bool]) - 是一个number.Number、bool值或数据类型为 `number <https://www.mindspore.cn/docs/zh-CN/r1.10/api_python/mindspore/mindspore.dtype.html#mindspore.dtype>`_ 或 `bool_ <https://www.mindspore.cn/docs/zh-CN/r1.10/api_python/mindspore/mindspore.dtype.html#mindspore.dtype>`_ 的Tensor。
