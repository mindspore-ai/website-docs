mindspore.dataset.vision.AutoAugmentPolicy

class mindspore.dataset.vision.AutoAugmentPolicy[源代码]

AutoAugment policy for different datasets.

Possible enumeration values are: AutoAugmentPolicy.IMAGENET, AutoAugmentPolicy.CIFAR10, AutoAugmentPolicy.SVHN.

Each policy contains 25 pairs of augmentation operations. When using AutoAugment, each image is randomly transformed with one of these operation pairs. Each pair has 2 different operations. The following shows all of these augmentation operations, including operation names with their probabilities and random params.
