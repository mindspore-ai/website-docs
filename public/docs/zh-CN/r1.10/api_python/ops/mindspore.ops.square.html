mindspore.ops.square¶

mindspore.ops.square(x)[源代码]¶

逐元素返回Tensor的平方。

\[y_i = x_i ^ 2\]
参数：

  • x (Tensor) - 输入Tensor的维度范围为[0,7]，类型为数值类型。

返回：

Tensor，具有与当前Tensor相同的数据类型和shape。

异常：

  • TypeError - x 不是Tensor。

支持平台：

Ascend GPU CPU

样例：

>>> x = Tensor(np.array([1.0, 2.0, 3.0]), mindspore.float32)
>>> output = ops.square(x)
>>> print(output)
[1. 4. 9.]
