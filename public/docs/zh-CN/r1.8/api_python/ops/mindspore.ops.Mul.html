mindspore.ops.Mul

class mindspore.ops.Mul[源代码]

两个Tensor逐元素相乘。

更多参考详见 mindspore.ops.mul()。

支持平台：

Ascend GPU CPU

样例：

>>> x = Tensor(np.array([1.0, 2.0, 3.0]), mindspore.float32)
>>> y = Tensor(np.array([4.0, 5.0, 6.0]), mindspore.float32)
>>> mul = ops.Mul()
>>> output = mul(x, y)
>>> print(output)
[ 4. 10. 18.]
