概述

mindspore.ops.adaptive_avg_pool3d

Ascend GPU CPU

mindspore.ops.conv2d

mindspore.ops.deformable_conv2d

Ascend GPU CPU

mindspore.ops.dropout2d

在训练期间，以服从伯努利分布的概率 p 随机将输入Tensor的某些通道归零。

Ascend GPU CPU

mindspore.ops.dropout3d

在训练期间，以服从伯努利分布的概率 p 随机将输入Tensor的某些通道归零。

Ascend CPU GPU

mindspore.ops.lrn

局部响应归一化操作LRN(Local Response Normalization)。

Ascend GPU CPU

mindspore.ops.max_pool3d

GPU

mindspore.ops.kl_div

计算输入 logits 和 labels 的KL散度。

Ascend GPU CPU

mindspore.ops.pad

根据参数 paddings 对输入进行填充。

mindspore.ops.padding

损失函数¶

接口名

概述

支持平台

mindspore.ops.binary_cross_entropy_with_logits

输入经过sigmoid激活函数后作为预测值，binary_cross_entropy_with_logits 计算预测值和目标值之间的二值交叉熵损失。

Ascend GPU CPU

mindspore.ops.cross_entropy

获取预测值和目标值之间的交叉熵损失。

Ascend GPU CPU

mindspore.ops.nll_loss

获取预测值和目标值之间的负对数似然损失。

Ascend GPU CPU

mindspore.ops.smooth_l1_loss

计算平滑L1损失，该L1损失函数有稳健性。

Ascend GPU CPU

接口名

概述

支持平台

mindspore.ops.mish

mindspore.ops.selu

激活函数selu（Scaled exponential Linear Unit）。

Ascend GPU CPU

mindspore.ops.soft_shrink

Soft Shrink激活函数，按输入元素计算输出。

Ascend CPU GPU

mindspore.ops.softsign

Ascend GPU CPU

mindspore.ops.softmax

Softmax函数。

mindspore.ops.tanh
