mindspore.nn.Top1CategoricalAccuracy

class mindspore.nn.Top1CategoricalAccuracy[源代码]

计算top-1分类正确率。此类是TopKCategoricalAccuracy的特殊类。有关更多详细信息，请参阅 TopKCategoricalAccuracy。

支持平台：

Ascend GPU CPU

样例：

>>> import numpy as np
>>> from mindspore import nn, Tensor
>>>
>>> x = Tensor(np.array([[0.2, 0.5, 0.3, 0.6, 0.2], [0.1, 0.35, 0.5, 0.2, 0.],
...         [0.9, 0.6, 0.2, 0.01, 0.3]]), mindspore.float32)
>>> y = Tensor(np.array([2, 0, 1]), mindspore.float32)
>>> topk = nn.Top1CategoricalAccuracy()
>>> topk.clear()
>>> topk.update(x, y)
>>> output = topk.eval()
>>> print(output)
0.0
