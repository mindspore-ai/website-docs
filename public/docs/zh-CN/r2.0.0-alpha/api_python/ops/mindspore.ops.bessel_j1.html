mindspore.ops.bessel_j1

mindspore.ops.bessel_j1(x)[源代码]

逐元素计算并返回输入Tensor的Bessel j1函数值。

参数：

  • x (Tensor) - 任意维度的Tensor。数据类型应为float16，float32或float64。

返回：

Tensor，shape和数据类型与 x 相同。

异常：

  • TypeError - x 不是Tensor。

  • TypeError - x 的数据类型不是float16，float32或float64。

支持平台：

GPU CPU

样例：

>>> x = Tensor(np.array([0.5, 1., 2., 4.]), mindspore.float32)
>>> output = ops.bessel_j1(x)
>>> print(output)
[0.24226846  0.44005059  0.57672481 -0.06604333]
