mindspore.ops.rank

mindspore.ops.rank(input_x)[源代码]

返回输入Tensor的秩，是一个零维的，数据类型为int32。Tensor的秩是确定Tensor中每个元素所需的索引数。

参数：

  • input_x (Tensor) - Tensor的shape为 \((x_1,x_2,...,x_R)\) 。数据类型为数值型。

返回：

Tensor，表示输入的秩，是一个零维的，数据类型为int32，即 \(R\) 。数据类型为int。

异常：

  • TypeError - 如果 input_x 不是Tensor。

支持平台：

Ascend GPU CPU

样例：

>>> input_tensor = Tensor(np.array([[2, 2], [2, 2]]), mindspore.float32)
>>> output = ops.rank(input_tensor)
>>> print(output)
2
>>> print(type(output))
<class 'int'>
