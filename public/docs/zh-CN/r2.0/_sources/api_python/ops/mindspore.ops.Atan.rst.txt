mindspore.ops.Atan
==================

.. py:class:: mindspore.ops.Atan

    逐元素计算输入Tensor的反正切值。

    更多细节请参考 :func:`mindspore.ops.atan` 。

    输入：
        - **x** (Tensor): 输入Tensor，shape: :math:`(N,*)` ，其中 :math:`*` 表示任何数量的附加维度。其轶应小于8。数据类型可为以下类型：float16或float32。

    输出：
        Tensor，shape和dtype与输入 `x` 相同。
