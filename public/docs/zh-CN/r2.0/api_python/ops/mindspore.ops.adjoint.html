mindspore.ops.adjoint

mindspore.ops.adjoint(x)[源代码]

逐元素计算Tensor的共轭，并转置最后两个维度。

参数：

  • x (Tensor) - 参与计算的Tensor。

返回：

Tensor，和 x 具有相同的dtype和shape。

异常：

  • TypeError：x 不是Tensor。

支持平台：

Ascend GPU CPU

样例：

>>> a = Tensor(np.array([[0. + 0.j, 1. + 1.j], [2. + 2.j, 3. + 3.j]]), mindspore.complex128)
>>> output = ops.adjoint(a)
>>> print(output)
[[0.-0.j 2.-2.j]
 [1.-1.j 3.-3.j]]
