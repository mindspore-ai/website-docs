mindspore.nn.LRScheduler

class mindspore.nn.LRScheduler(optimizer, last_epoch=- 1, verbose=False)[源代码]

动态学习率的基类。

警告

这是一个实验性的动态学习率模块，需要和 实验性优化器 下的接口配合使用。

参数：

  • optimizer (mindspore.nn.optim_ex.Optimizer) - 优化器实例。

  • last_epoch (int，可选) - epoch/step数。默认值： -1。

  • verbose (bool，可选) - 是否打印学习率。默认值： False。

异常：

  • TypeError - optimizer 不是优化器。

  • TypeError - last_epoch 小于-1。

  • ValueError - verbose 不是布尔类型。

支持平台：

Ascend GPU CPU
