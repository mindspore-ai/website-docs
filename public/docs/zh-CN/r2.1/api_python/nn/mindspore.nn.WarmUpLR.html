mindspore.nn.WarmUpLRпғҒ

class mindspore.nn.WarmUpLR(learning_rate, warmup_steps)[жәҗд»Јз Ғ]пғҒ

йў„зғӯеӯҰд№ зҺҮгҖӮ

еҜ№дәҺеҪ“еүҚstepпјҢи®Ўз®—еӯҰд№ зҺҮзҡ„е…¬ејҸдёәпјҡ

\[warmup\_learning\_rate = learning\_rate * tmp\_step / warmup\_steps\]

е…¶дёӯпјҢ

\[tmp\_step= \min(current\_step, warmup\_steps)\]
еҸӮж•°пјҡ

  • learning_rate (float) - еӯҰд№ зҺҮзҡ„еҲқе§ӢеҖјгҖӮ

  • warmup_steps (int) - еӯҰд№ зҺҮwarmupзҡ„stepж•°гҖӮ

иҫ“е…Ҙпјҡ

  • global_step (Tensor) - еҪ“еүҚstepж•°пјҢеҚіcurrent_stepпјҢshapeдёә \(()\)гҖӮ

иҫ“еҮәпјҡ

ж ҮйҮҸTensorгҖӮеҪ“еүҚstepзҡ„еӯҰд№ зҺҮеҖјгҖӮ

ејӮеёёпјҡ

  • TypeError - learning_rate дёҚжҳҜfloatгҖӮ

  • TypeError - warmup_steps дёҚжҳҜintгҖӮ

  • ValueError - warmup_steps е°ҸдәҺ1гҖӮ

  • ValueError - learning_rate е°ҸдәҺжҲ–зӯүдәҺ0гҖӮ

ж”ҜжҢҒе№іеҸ°пјҡ

Ascend GPU CPU

ж ·дҫӢпјҡ

>>> import mindspore
>>> from mindspore import Tensor, nn
>>>
>>> learning_rate = 0.1
>>> warmup_steps = 2
>>> global_step = Tensor(2, mindspore.int32)
>>> warmup_lr = nn.WarmUpLR(learning_rate, warmup_steps)
>>> lr = warmup_lr(global_step)
>>> net = nn.Dense(2, 3)
>>> optim = nn.SGD(net.trainable_params(), learning_rate=lr)
