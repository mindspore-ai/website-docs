mindspore.ops.binary_cross_entropy_with_logitsпғҒ

mindspore.ops.binary_cross_entropy_with_logits(logits, label, weight, pos_weight, reduction='mean')[жәҗд»Јз Ғ]пғҒ

иҫ“е…Ҙз»ҸиҝҮsigmoidжҝҖжҙ»еҮҪж•°еҗҺдҪңдёәйў„жөӢеҖјпјҢbinary_cross_entropy_with_logits и®Ўз®—йў„жөӢеҖје’Ңзӣ®ж ҮеҖјд№Ӣй—ҙзҡ„дәҢеҖјдәӨеҸүзҶөжҚҹеӨұгҖӮ

е°Ҷиҫ“е…Ҙ logits и®ҫзҪ®дёә \(X\) пјҢиҫ“е…Ҙ labels и®ҫзҪ®дёә \(Y\) пјҢиҫ“е…Ҙ weight и®ҫзҪ®дёә \(W\) пјҢиҫ“еҮәи®ҫзҪ®дёә \(L\) гҖӮеҲҷпјҢ

\[\begin{split}\begin{array}{ll} \\ p_{ij} = sigmoid(X_{ij}) = \frac{1}{1 + e^{-X_{ij}}} \\ L_{ij} = -[Y_{ij}log(p_{ij}) + (1 - Y_{ij})log(1 - p_{ij})] \end{array}\end{split}\]

\(i\) иЎЁзӨә \(i^{th}\) ж ·дҫӢпјҢ \(j\) иЎЁзӨәзұ»еҲ«гҖӮеҲҷпјҢ

\[\begin{split}\ell(x, y) = \begin{cases} L, & \text{if reduction} = \text{'none';}\\ \operatorname{mean}(L), & \text{if reduction} = \text{'mean';}\\ \operatorname{sum}(L), & \text{if reduction} = \text{'sum'.} \end{cases}\end{split}\]

\(\ell\) иЎЁзӨәи®Ўз®—жҚҹеӨұзҡ„ж–№жі•гҖӮжңүдёүз§Қж–№жі•пјҡз¬¬дёҖз§Қж–№жі•жҳҜзӣҙжҺҘжҸҗдҫӣжҚҹеӨұеҖјпјҢз¬¬дәҢз§Қж–№жі•жҳҜи®Ўз®—жүҖжңүжҚҹеӨұзҡ„е№іеқҮеҖјпјҢз¬¬дёүз§Қж–№жі•жҳҜи®Ўз®—жүҖжңүжҚҹеӨұзҡ„жҖ»е’ҢгҖӮ

иҜҘз®—еӯҗдјҡе°Ҷиҫ“еҮәд№ҳд»Ҙзӣёеә”зҡ„жқғйҮҚгҖӮ \(weight\) иЎЁзӨәдёҖдёӘbatchдёӯзҡ„жҜҸжқЎж•°жҚ®еҲҶй…ҚдёҚеҗҢзҡ„жқғйҮҚпјҢ \(pos_weight\) дёәжҜҸдёӘзұ»еҲ«зҡ„жӯЈдҫӢеӯҗж·»еҠ зӣёеә”зҡ„жқғйҮҚгҖӮ

жӯӨеӨ–пјҢе®ғеҸҜд»ҘйҖҡиҝҮеҗ‘жӯЈдҫӢж·»еҠ жқғйҮҚжқҘжқғиЎЎеҸ¬еӣһзҺҮе’ҢзІҫеәҰгҖӮ еңЁеӨҡж ҮзӯҫеҲҶзұ»зҡ„жғ…еҶөдёӢпјҢжҚҹеӨұеҸҜд»ҘжҸҸиҝ°дёәпјҡ

\[\begin{split}\begin{array}{ll} \\ p_{ij,c} = sigmoid(X_{ij,c}) = \frac{1}{1 + e^{-X_{ij,c}}} \\ L_{ij,c} = -[P_{c}Y_{ij,c} * log(p_{ij,c}) + (1 - Y_{ij,c})log(1 - p_{ij,c})] \end{array}\end{split}\]

е…¶дёӯ c жҳҜзұ»еҲ«ж•°зӣ®пјҲc>1 иЎЁзӨәеӨҡж ҮзӯҫдәҢе…ғеҲҶзұ»пјҢc=1 иЎЁзӨәеҚ•ж ҮзӯҫдәҢе…ғеҲҶзұ»пјүпјҢn жҳҜжү№ж¬Ўдёӯж ·жң¬зҡ„ж•°йҮҸпјҢ\(P_c\) жҳҜ з¬¬cзұ»жӯЈдҫӢзҡ„жқғйҮҚгҖӮ \(P_c>1\) еўһеӨ§еҸ¬еӣһзҺҮ, \(P_c<1\) еўһеӨ§зІҫеәҰгҖӮ

еҸӮж•°пјҡ

  • logits (Tensor) - иҫ“е…Ҙйў„жөӢеҖјгҖӮе…¶ж•°жҚ®зұ»еһӢдёәfloat16жҲ–float32гҖӮ

  • label (Tensor) - иҫ“е…Ҙзӣ®ж ҮеҖјпјҢshapeдёҺ logits зӣёеҗҢгҖӮж•°жҚ®зұ»еһӢдёәfloat16жҲ–float32гҖӮ

  • weight (Tensor) - жҢҮе®ҡжҜҸдёӘжү№ж¬ЎдәҢеҖјдәӨеҸүзҶөзҡ„жқғйҮҚгҖӮж”ҜжҢҒе№ҝж’ӯпјҢдҪҝе…¶shapeдёҺ logits зҡ„shapeдҝқжҢҒдёҖиҮҙгҖӮж•°жҚ®зұ»еһӢеҝ…йЎ»дёәfloat16жҲ–float32гҖӮ

  • pos_weight (Tensor) - жҢҮе®ҡжӯЈзұ»зҡ„жқғйҮҚгҖӮжҳҜдёҖдёӘй•ҝеәҰзӯүдәҺеҲҶзұ»ж•°зҡ„еҗ‘йҮҸгҖӮж”ҜжҢҒе№ҝж’ӯпјҢдҪҝе…¶shapeдёҺ logits зҡ„shapeдҝқжҢҒдёҖиҮҙгҖӮж•°жҚ®зұ»еһӢеҝ…йЎ»дёәfloat16жҲ–float32гҖӮ

  • reduction (str) - жҢҮе®ҡз”ЁдәҺиҫ“еҮәз»“жһңзҡ„и®Ўз®—ж–№ејҸгҖӮеҸ–еҖјдёә 'mean' гҖҒ 'sum' жҲ– 'none' пјҢдёҚеҢәеҲҶеӨ§е°ҸеҶҷгҖӮеҰӮжһң 'none' пјҢеҲҷдёҚжү§иЎҢ reduction гҖӮй»ҳи®ӨеҖјпјҡ 'mean' гҖӮ

иҝ”еӣһпјҡ

TensorжҲ–ScalarпјҢеҰӮжһң reduction дёә вҖҳnoneвҖҷ пјҢеҲҷдёәshapeе’Ңж•°жҚ®зұ»еһӢдёҺиҫ“е…Ҙ logits зӣёеҗҢзҡ„TensorгҖӮеҗҰеҲҷпјҢиҫ“еҮәдёәScalarгҖӮ

ејӮеёёпјҡ

  • TypeError - иҫ“е…Ҙ logits пјҢ label пјҢ weight пјҢ pos_weight дёҚдёәTensorгҖӮ

  • TypeError - иҫ“е…Ҙ logits пјҢ label пјҢ weight пјҢ pos_weight зҡ„ж•°жҚ®зұ»еһӢж—ўдёҚжҳҜfloat16д№ҹдёҚжҳҜfloat32гҖӮ

  • TypeError - иҫ“е…Ҙ reduction иҫ“е…Ҙж•°жҚ®зұ»еһӢдёҚдёәstringгҖӮ

  • ValueError - weight жҲ– pos_weight дёҚиғҪе№ҝж’ӯеҲ°shapeдёә logits зҡ„TensorгҖӮ

  • ValueError - reduction дёҚдёә вҖҳnoneвҖҷ гҖҒ вҖҳmeanвҖҷ жҲ– вҖҳsumвҖҷ гҖӮ

>>> import mindspore
>>> import numpy as np
>>> from mindspore import Tensor, ops
>>> logits = Tensor(np.array([[-0.8, 1.2, 0.7], [-0.1, -0.4, 0.7]]), mindspore.float32)
>>> label = Tensor(np.array([[0.3, 0.8, 1.2], [-0.6, 0.1, 2.2]]), mindspore.float32)
>>> weight = Tensor(np.array([1.0, 1.0, 1.0]), mindspore.float32)
>>> pos_weight = Tensor(np.array([1.0, 1.0, 1.0]), mindspore.float32)
>>> output = ops.binary_cross_entropy_with_logits(logits, label, weight, pos_weight)
>>> print(output)
0.3463612
