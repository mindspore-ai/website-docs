.. py:class:: mindspore.dataset.text.JiebaMode

    Jieba分词器可使用的分词算法。

    可选值如下：

    - **JiebaMode.MIX** - 最大概率法（Max Probability）和隐式马尔可夫模型（Hidden Markov Model）混合分词。
    - **JiebaMode.MP** - 最大概率法（Max Probability）分词。
    - **JiebaMode.HMM** - 隐式马尔可夫模型（Hidden Markov Model）分词。
