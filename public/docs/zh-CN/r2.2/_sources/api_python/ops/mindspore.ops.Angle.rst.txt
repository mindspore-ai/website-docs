.. py:class:: mindspore.ops.Angle

    逐元素计算复数Tensor的辐角。

    .. warning::
        这是一个实验性API，后续可能修改或删除。

    更多参考详见 :func:`mindspore.ops.angle` 。

    输入：
        - **input** (Tensor) - 输入Tensor。支持类型：complex64、complex128。

    输出：
        Tensor，类型为float32或float64，shape与输入相同。
