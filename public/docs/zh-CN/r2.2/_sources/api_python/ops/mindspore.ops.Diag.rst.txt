.. py:class:: mindspore.ops.Diag

    用给定的对角线值构造对角线Tensor。

    .. warning::
        这是一个实验性API，后续可能修改或删除。

    更多参考详见 :func:`mindspore.ops.diag`。

    输入：
        - **input_x** (Tensor) - 输入Tensor。

    输出：
        Tensor，数据类型与 `input_x` 一致。
