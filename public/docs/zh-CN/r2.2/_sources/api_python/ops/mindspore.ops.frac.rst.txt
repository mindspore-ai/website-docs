.. py:function:: mindspore.ops.frac(x)

    计算 x 中每个元素的小数部分。

    参数：
        - **x** (Tensor) - 输入Tensor。

    返回：
        Tensor， `x` 的frac计算结果。

    异常：
        - **TypeError** - 如果 `x` 不是Tensor。
