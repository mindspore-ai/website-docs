.. py:method:: mindspore.Tensor.type_as(other)

    将Tensor的dtype转换成 `other` 的dtype。

    参数：
        - **other** (Tensor) - 返回的Tensor和 `other` 具有相同的dtype。

    返回：
        Tensor，和 `other` 有相同的dtype。
