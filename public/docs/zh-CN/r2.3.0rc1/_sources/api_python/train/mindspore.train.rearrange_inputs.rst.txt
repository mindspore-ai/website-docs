.. py:function:: mindspore.train.rearrange_inputs(func)

    此装饰器用于根据类的 `indexes` 属性对输入重新排列。

    此装饰器目前用于 :class:`mindspore.train.Metric` 类的 `update` 方法。

    参数：
        - **func** (Callable) - 要装饰的候选函数，其输入将被重新排列。

    返回：
        Callable，用于在函数之间调换输入。
