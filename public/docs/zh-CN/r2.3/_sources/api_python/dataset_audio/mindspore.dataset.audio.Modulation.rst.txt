.. py:class:: mindspore.dataset.audio.Modulation

    调制类型。

    可选的枚举值包括： ``Modulation.SINUSOIDAL`` 、 ``Modulation.TRIANGULAR`` 。

    - **Modulation.SINUSOIDAL** - 表示输入调制类型为正弦。
    - **Modulation.TRIANGULAR** - 表示输入调制类型为三角形。
