.. py:method:: mindspore.Tensor.reshape_as(other)

    不改变数据的情况下，将Tensor的shape改为 `other` 的shape。

    参数：
        - **other** (Tensor) - 计算结果的shape与 `other` 的shape一致。

    返回：
        Tensor，与 `other` 的shape一致。
