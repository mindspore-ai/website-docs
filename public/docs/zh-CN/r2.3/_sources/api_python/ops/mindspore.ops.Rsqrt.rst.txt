.. py:class:: mindspore.ops.Rsqrt

    逐元素计算输入Tensor平方根的倒数。

    更多参考详见 :func:`mindspore.ops.rsqrt`。

    输入：
        - **x** (Tensor) - 输入Tensor，其每个元素都为非负，若某个元素为负，计算结果为nan。

    输出：
        Tensor，数据类型和shape与 `x` 相同。
