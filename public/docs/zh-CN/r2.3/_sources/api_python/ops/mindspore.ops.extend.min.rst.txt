.. py:function:: mindspore.ops.extend.min(input, dim, keepdim=False)

    在给定轴上计算输入Tensor的最小值。并且返回最小值和索引。

    参考：:class:`mindspore.ops.ArgMinWithValue`。

    参数：
        - **input** (Tensor) - 输入任意维度的Tensor。不支持复数类型。
        - **axis** (int) - 指定计算维度。
        - **keepdims** (bool) - 表示是否减少维度，如果为 ``True`` ，输出将与输入保持相同的维度；如果为 ``False`` ，输出将减少维度。默认值： ``False`` 。

    返回：
        tuple (Tensor)，表示2个Tensor组成的tuple，包含对应的索引和输入Tensor的最小值。

        - **values** (Tensor) - 输入Tensor的最小值，其shape与索引相同。
        - **index** (Tensor) - 输入Tensor最小值的索引。如果 `keepdims` 为True，则输出Tensor的shape为 :math:`(input_1, input_2, ..., input_{axis-1}, 1, input_{axis+1}, ..., input_N)` 。否则，shape为 :math:`(input_1, input_2, ..., input_{axis-1}, input_{axis+1}, ..., input_N)` 。
