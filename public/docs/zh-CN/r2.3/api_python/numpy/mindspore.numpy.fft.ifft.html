mindspore.numpy.fft.ifft

mindspore.numpy.fft.ifft(a, n=None, axis=- 1, norm=None)[源代码]

Calculates the inverse of fft().

Refer to mindspore.ops.ifft() for more details. The difference is that a corresponds to input and axis corresponds to dim.

参数

  • a (Tensor) –

    The input tensor. Supported dtypes:

    • Ascend/CPU: int16, int32, int64, float16, float32, float64, complex64, complex128.

  • n (int, optional) – Length of the transformed dim of the result.

  • n – Signal length. If given, the input will either be zero-padded or trimmed to this length before computing ifft. Default: None , which does not need to process a.

  • axis (int, optional) – Axis over which to compute the ifft. Default: -1 , which means the last axis of a is used.

  • norm (string, optional) –

    Normalization mode. Default: None that means "backward" . Three modes are defined as,

    • "backward" (no normalization).

    • "forward" (normalize by \(1*n\)).

    • "ortho" (normalize by \(1*\sqrt{n}\)).

返回

Tensor, The result of ifft() function. The default is the same shape as a. If n is given, the size of the axis is changed to n. When the a is int16, int32, int64, float16, float32, complex64, the return value type is complex64. When the a is float64 or complex128, the return value type is complex128.

Supported Platforms:

Ascend CPU

样例

>>> import mindspore.numpy as np
>>> input = np.array([ 1.6243454, -0.6117564, -0.5281718, -1.0729686])
>>> np.fft.ifft(input)
Tensor(shape=[4], dtype=Complex64, value= [-0.147138+0j, 0.538129+0.115303j, 0.695225+0j, 0.538129-0.115303j])
