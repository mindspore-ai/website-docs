isUseElb

public boolean isUseElb()

是否模拟弹性负载均衡，即客户端将请求随机发给一定范围内的server地址。

  • 返回值

    boolean类型，true代表客户端会将请求随机发给一定范围内的server地址， false客户端的请求会发给固定的server地址。
