定义联邦学习云侧任务。

参数：

  • yaml_config (str) - yaml文件路径。更多细节见 yaml配置说明。

  • http_server_address (str) - 用于通信的http服务器地址。

  • tcp_server_ip (str) - 用于通信的tcp服务器地址。默认值：127.0.0.1。

  • checkpoint_dir (str) - 存储权重的路径。默认值：”./fl_ckpt/”。

  • ssl_config (Union(None, SSLConfig)) - ssl配置项。默认值：None。

样例：

>>> job = FLServerJob(yaml_config=yaml_config, http_server_address=http_server_address,
...                   tcp_server_ip=tcp_server_ip, checkpoint_dir=checkpoint_dir)
>>> job.run()
