class SubsetSampler : public mindspore::dataset::Sampler

A class to represent a Subset Sampler in the data pipeline.

Note

Samples the elements from a sequence of indices.

Subclassed by mindspore::dataset::SubsetRandomSampler

Public Functions

SubsetSampler(const std::vector<int64_t> &indices, int64_t num_samples = 0)

Constructor.

Parameters

  • [in] indices: A vector sequence of indices.

  • [in] num_samples: The number of samples to draw (default=0, return all samples).

Example

/* creates a SubsetSampler, will sample from the provided indices */
std::string folder_path = "/path/to/image/folder";
std::shared_ptr<Dataset> ds = ImageFolder(folder_path, false, std::make_shared<SubsetSampler>({0, 2, 5}));
