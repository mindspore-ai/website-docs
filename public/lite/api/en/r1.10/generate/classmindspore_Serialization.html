class Serialization

The Serialization class is used to summarize methods for reading and writing model files.

Public Static Functions

Status Load(const void *model_data, size_t data_size, ModelType model_type, Graph *graph, const Key &dec_key = {}, const std::string &dec_mode = kDecModeAesGcm)

Loads a model file from memory buffer.

Parameters

  • [in] model_data: A buffer filled by model file.

  • [in] data_size: The size of the buffer.

  • [in] model_type: The Type of model file, options are ModelType::kMindIR, ModelType::kOM.

  • [out] graph: The output parameter, an object saves graph data.

  • [in] dec_key: The decryption key, key length is 16, 24, or 32. Not supported on MindSpore Lite.

  • [in] dec_mode: The decryption mode, optional options are AES-GCM, AES-CBC. Not supported on MindSpore Lite.

Return

Status.
