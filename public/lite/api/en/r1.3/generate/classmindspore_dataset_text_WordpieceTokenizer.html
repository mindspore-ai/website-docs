class WordpieceTokenizer : public mindspore::dataset::TensorTransform¶

Tokenize scalar token or 1-D tokens to 1-D sub-word tokens.

Public Functions

WordpieceTokenizer(const std::shared_ptr<Vocab> &vocab, const std::string &suffix_indicator = "##", int32_t max_bytes_per_token = 100, const std::string &unknown_token = "[UNK]", bool with_offsets = false)¶

Constructor.

Parameters

  • [in] vocab: A Vocab object.

  • [in] suffix_indicator: This parameter is used to show that the sub-word is the last part of a word (default=’##’).

  • [in] max_bytes_per_token: Tokens exceeding this length will not be further split (default=100).

  • [in] unknown_token: When a token cannot be found, return the token directly if ‘unknown_token’ is an empty string, else return the specified string (default=’[UNK]’).

  • [in] with_offsets: whether to output offsets of tokens (default=false).
