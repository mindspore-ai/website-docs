enum class mindspore::dataset::InterpolationMode

Possible options for interpolation method.

Values:

enumerator kLinear

Interpolation method is linear interpolation.

enumerator kNearestNeighbour

Interpolation method is nearest-neighbor interpolation.

enumerator kCubic

Interpolation method is bicubic interpolation.

enumerator kArea

Interpolation method is pixel area interpolation.

enumerator kCubicPil

Interpolation method is bicubic interpolation like implemented in pillow.
