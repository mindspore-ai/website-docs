Function mindspore::dataset::GaussianBlur¶

  • Defined in File image_process.h

Function Documentation¶

bool mindspore::dataset::GaussianBlur(const LiteMat &src, LiteMat &dst, const std::vector<int> &ksize, double sigmaX, double sigmaY = 0.f, PaddBorderType pad_type = PaddBorderType::PADD_BORDER_DEFAULT)¶

Filter the image by a Gaussian kernel.

Parameters

  • [in] src: LiteMat image to be processed. Only LiteMat of type UINT8 is supported now.

  • [in] dst: LiteMat image after processing.

  • [in] ksize: The size of Gaussian kernel. It should be a vector of size 2 as {kernel_x, kernel_y}, both value of which should be positive and odd.

  • [in] sigmaX: The Gaussian kernel standard deviation of width. It should be a positive value.

  • [in] sigmaY: The Gaussian kernel standard deviation of height (default=0.f). It should be a positive value, or will use the value of sigmaX.

  • [in] pad_type: The padding type used while filtering (default=PaddBorderType::PADD_BORDER_DEFAULT).
