void Init(const std::vector<int64_t> &min_sizes, const std::vector<int64_t> &max_sizes, const std::vector<float> &aspect_ratios, const std::vector<float> &variances, const int64_t image_size_w, const int64_t image_size_h, const float step_w, const float step_h, const bool clip, const bool flip, const float offset)

Method to init the op’s attributes.

Parameters

  • [in] min_sizes: Define the minimum side length of square boxes, can be multiple.

  • [in] max_sizes: Define the maximum side length of square boxes as sqrt(min_size * max_size), can be multiple.

  • [in] aspect_ratios: Define the aspect ratios of generated boxes. For each aspect_ratio, the width and height are min_size * sqrt(aspect_ratio) and min_size / sqrt(aspect_ratio) respectively.

  • [in] variances: Define variances for adjusting the prior boxes.

  • [in] image_size_w: Define the width of the image.

  • [in] image_size_h: Define the height of the image.

  • [in] step_w: Define the ratio of the image width and the feature map width.

  • [in] step_h: Define the ratio of the image height and the feature map height.

  • [in] clip: Define whether clip the prior boxes to [0.0, 1.0].

  • [in] flip: Define whether flip aspect ratios. If true, with an aspect ratio r, ratio 1.0/r will be generated.

  • [in] offset: Define the offset to the zero points of width and height.
