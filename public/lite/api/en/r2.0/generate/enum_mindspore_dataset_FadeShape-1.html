enum mindspore::dataset::FadeShape¶

Possible options for fade shape.

Values:

kLinear = 0¶

Fade shape is linear mode.

kExponential = 1¶

Fade shape is exponential mode.

kLogarithmic = 2¶

Fade shape is logarithmic mode.

kQuarterSine = 3¶

Fade shape is quarter_sine mode.

kHalfSine = 4¶

Fade shape is half_sine mode.
