class Slice¶

Slice object used in SliceOption.

Public Functions

Slice()¶

Constructor, with start, stop and step default to 0.

Slice(dsize_t start, dsize_t stop, dsize_t step)¶

Constructor.

Parameters

  • [in] start: Starting integer specifying where to start the slicing.

  • [in] stop: Ending integer specifying where to stop the slicing.

  • [in] step: An integer specifying the step of the slicing.

Example

/* Slice dimension from 2 to 10 with step 2. */
Slice(0, 10, 2);
Slice(dsize_t start, dsize_t stop)¶

Constructor, with step=1.

Parameters

  • [in] start: Starting integer specifying where to start the slicing.

  • [in] stop: Ending integer specifying where to stop the slicing.

Example

/* Slice dimension from 5 to 10 with step 1. */
Slice(5, 10);

Constructor, with start=0 and step=1.

Parameters

  • [in] stop: Ending integer specifying where to stop the slicing.
