class SubsetRandomSampler : public mindspore::dataset::SubsetSampler¶

A class to represent a Subset Random Sampler in the data pipeline.

Note

Samples the elements randomly from a sequence of indices.

Public Functions

SubsetRandomSampler(const std::vector<int64_t> &indices, int64_t num_samples = 0)¶

Constructor.

Parameters

  • [in] indices: A vector sequence of indices.

  • [in] num_samples: The number of samples to draw (default=0, return all samples).

Example

/* create a SubsetRandomSampler, will sample from the provided indices */
std::string folder_path = "/path/to/image/folder";
std::shared_ptr<Dataset> ds = ImageFolder(folder_path, false, std::make_shared<SubsetRandomSampler>({2, 7}));
