class RandomApply : public mindspore::dataset::TensorTransform¶

Randomly perform a series of transforms with a given probability.

Public Functions

RandomApply(const std::vector<TensorTransform *> &transforms, double prob = 0.5)¶

Constructor.

Parameters

  • [in] transforms: A vector of raw pointers to TensorTransform objects to be applied.

  • [in] prob: The probability to apply the transformation list (default=0.5).

Example

/* Define operations */
auto resize_op(new vision::Resize({30, 30}));
auto center_crop_op(new vision::CenterCrop({16, 16}));
auto random_op(new transforms::RandomApply({resize_op, center_crop_op}));

/* dataset is an instance of Dataset object */
dataset = dataset->Map({random_op},   // operations
                       {"image"});    // input columns
