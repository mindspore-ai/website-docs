Function mindspore::dataset::Album

  • Defined in File datasets.h

Function Documentation

inline std::shared_ptr<AlbumDataset> mindspore::dataset::Album(const std::string &dataset_dir, const std::string &data_schema, const std::vector<std::string> &column_names, bool decode, const std::reference_wrapper<Sampler> sampler, const std::shared_ptr<DatasetCache> &cache = nullptr)

Function to create an AlbumDataset.

说明

The generated dataset is specified through setting a schema

参数

  • dataset_dir – [in] Path to the root directory that contains the dataset

  • data_schema – [in] Path to dataset schema file

  • column_names – [in] Column names used to specify columns to load

  • decode – [in] the option to decode the images in dataset

  • sampler – [in] Sampler object used to choose samples from the dataset.

  • cache – [in] Tensor cache to use. (default=nullptr which means no cache is used).

返回

Shared pointer to the current Dataset
