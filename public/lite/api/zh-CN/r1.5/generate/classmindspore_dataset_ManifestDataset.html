class ManifestDataset : public mindspore::dataset::Dataset

A source dataset for reading and parsing Manifest dataset.

Public Functions

ManifestDataset(const std::vector<char> &dataset_file, const std::vector<char> &usage, const std::shared_ptr<Sampler> &sampler, const std::map<std::vector<char>, int32_t> &class_indexing, bool decode, const std::shared_ptr<DatasetCache> &cache)

Constructor of ManifestDataset.

Parameters

  • dataset_file – [in] The dataset file to be read.

  • usage – [in] Part of dataset of ManifestDataset, can be “train”, “eval” or “inference” data (default=”train”).

  • sampler – [in] Shared pointer to a sampler object used to choose samples from the dataset. If sampler is not given, a RandomSampler will be used to randomly iterate the entire dataset (default = RandomSampler()).

  • class_indexing – [in] A str-to-int mapping from label name to index (default={}, the folder names will be sorted alphabetically and each class will be given a unique index starting from 0).

  • decode – [in] Decode the images after reading (default=false).

  • cache – [in] Tensor cache to use (default=nullptr which means no cache is used).
