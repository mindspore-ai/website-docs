Function mindspore::dataset::Cifar10¶

  • Defined in File datasets.h

Function Documentation¶

std::shared_ptr<Cifar10Dataset> mindspore::dataset::Cifar10(const std::string &dataset_dir, const std::string &usage, const std::reference_wrapper<Sampler> sampler, const std::shared_ptr<DatasetCache> &cache = nullptr)

Function to create a Cifar10 Dataset.

Note

The generated dataset has two columns [“image”, “label”].

Parameters

  • [in] dataset_dir: Path to the root directory that contains the dataset.

  • [in] usage: Part of dataset of CIFAR10, can be “train”, “test” or “all”.

  • [in] sampler: Sampler object used to choose samples from the dataset.

  • [in] cache: Tensor cache to use (default=nullptr which means no cache is used).

Return

Shared pointer to the Cifar10Dataset.
