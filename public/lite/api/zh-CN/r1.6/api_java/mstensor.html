public static MSTensor createTensor(String tensorName, int dataType, int[] tensorShape, ByteBuffer buffer)

    • dataType: 张量数据类型。

    • tensorShape: 张量形状。

    • buffer: 张量数据。

    MindSpore MSTensor。
