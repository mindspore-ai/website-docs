class VOCDataset : public mindspore::dataset::Dataset¶

A source dataset for reading and parsing VOC dataset.

Public Functions

VOCDataset(const std::vector<char> &dataset_dir, const std::vector<char> &task, const std::vector<char> &usage, const std::map<std::vector<char>, int32_t> &class_indexing, bool decode, const std::shared_ptr<Sampler> &sampler, const std::shared_ptr<DatasetCache> &cache, bool extra_metadata)¶

Constructor of VOCDataset.

Parameters

  • [in] dataset_dir: Path to the root directory that contains the dataset.

  • [in] task: Set the task type of reading voc data, now only support “Segmentation” or “Detection”.

  • [in] usage: The type of data list text file to be read (default = “train”).

  • [in] class_indexing: A str-to-int mapping from label name to index, only valid in “Detection” task.

  • [in] decode: Decode the images after reading.

  • [in] sampler: Shared pointer to a sampler object used to choose samples from the dataset. If sampler is not given, a RandomSampler will be used to randomly iterate the entire dataset (default = RandomSampler()).

  • [in] cache: Tensor cache to use (default=nullptr which means no cache is used).

  • [in] extra_metadata: Flag to add extra meta-data to row (default=false).
