Function mindspore::dataset::Mnist¶

  • Defined in File datasets.h

Function Documentation¶

std::shared_ptr<MnistDataset> mindspore::dataset::Mnist(const std::string &dataset_dir, const std::string &usage, const Sampler *sampler, const std::shared_ptr<DatasetCache> &cache = nullptr)

Function to create a MnistDataset.

Note

The generated dataset has two columns [“image”, “label”].

Parameters

  • [in] dataset_dir: Path to the root directory that contains the dataset.

  • [in] usage: Part of dataset of MNIST, can be “train”, “test” or “all”.

  • [in] sampler: Raw pointer to a sampler object used to choose samples from the dataset.

  • [in] cache: Tensor cache to use (default=nullptr which means no cache is used).

Return

Shared pointer to the MnistDataset.
