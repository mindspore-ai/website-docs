Function mindspore::dataset::Places365¶

Function Documentation¶

std::shared_ptr<Places365Dataset> mindspore::dataset::Places365(const std::string &dataset_dir, const std::string &usage, const bool small, const bool decode, const Sampler *sampler, const std::shared_ptr<DatasetCache> &cache = nullptr)

Function to create a Places365Dataset.

Note

The generated dataset has two columns [“image”, “label”].

Parameters

  • [in] dataset_dir: Path to the root directory that contains the dataset.

  • [in] usage: Dataset splits of Places365, can be train-standard, train-challenge or val.

  • [in] small: Use the small images instead of the high resolution ones.

  • [in] decode: Decode the images after reading.

  • [in] sampler: Raw pointer to a sampler object used to choose samples from the dataset.

  • [in] cache: Tensor cache to use. (default=nullptr which means no cache is used).

Return

Shared pointer to the current Places365Dataset.
