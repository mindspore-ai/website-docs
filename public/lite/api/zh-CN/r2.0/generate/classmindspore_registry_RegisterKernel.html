class RegisterKernel¶

RegisterKernel Defined registration of kernel.

Public Static Functions

Status RegKernel(const std::string &arch, const std::string &provider, DataType data_type, int type, const CreateKernel creator)¶

Static method to register kernel which is correspondng to an ordinary op.

Parameters

  • [in] arch: Define deviceType, such as CPU.

  • [in] provider: Define the identification of user.

  • [in] data_type: Define kernel’s input data type.

  • [in] type: Define the ordinary op type.

  • [in] creator: Define a function pointer to create a kernel.

Return

Status as a status identification of registering.

Static method to register kernel which is corresponding to custom op.

Parameters

  • [in] arch: Define deviceType, such as CPU.

  • [in] provider: Define the identification of user.

  • [in] data_type: Define kernel’s input data type.

  • [in] type: Define the concrete type of a custom op.

Return
