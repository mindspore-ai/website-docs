mindearth.data.RadarData

class mindearth.data.RadarData(data_params, run_mode='train')[source]

This class is used to process dgmr radar data, and is used to generate the dataset generator supported by MindSpore. This class inherits the Data class.

Parameters

  • data_params (dict) – dataset-related configuration of the model.

  • run_mode (str, optional) – whether the dataset is used for training, evaluation or testing. Supports [“train”, “test”, “valid”]. Default: ‘train’.

Supported Platforms:

Ascend GPU

Examples

>>> from mindearth.data import RadarData
>>> data_params = {
...     'name': 'radar',
...     'root_dir': './dataset',
...     'batch_size': 4,
...     'num_workers': 1,
...     't_out_train': '',
... }
>>> dataset_generator = RadarData(data_params)
