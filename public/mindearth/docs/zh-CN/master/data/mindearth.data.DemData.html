mindearth.data.DemData

class mindearth.data.DemData(data_params, run_mode='train')[源代码]

DemData类通过MindSpore框架处理Dem数据集生成数据生成器。DemData类继承了Data类。

参数：

  • data_params (dict) - 模型中的相关数据参数。

  • run_mode (str, 可选) - 决定数据集用于训练、验证还是测试。支持 'train', 'test', 'valid'。默认值： 'train'。

支持平台：

Ascend GPU

样例：

>>> from mindearth.data import DemData
>>> data_params = {
...     'name': 'nasadem',
...     'root_dir': './dataset',
...     'patch_size': 32,
...     'batch_size': 64,
...     'epoch_size': 10,
...     'num_workers': 1,
...     't_out_train': '',
... }
>>> dataset_generator = DemData(data_params)
