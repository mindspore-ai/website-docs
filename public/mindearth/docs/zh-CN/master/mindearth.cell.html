mindearth.cell.AFNONet

AFNO是一个基于傅立叶神经算子（FNO）并结合Vision Transformer结构的深度学习模型。

Ascend GPU

mindearth.cell.DEMNet

DEM超分辨率模型基于深度残差网络和迁移学习技术。

Ascend GPU

mindearth.cell.DgmrDiscriminator

Dgmr判别器基于时间判别器和空间判别器，其中包含深度残差块。

Ascend GPU

mindearth.cell.DgmrGenerator

Dgmr 生成器基于 Conditional_Stack、Latent_Stack、Upsample_Stack 和 ConvGRU，其中包含深度残差块。

Ascend GPU

mindearth.cell.GraphCastNet

GraphCast 基于一种新颖的基于图神经网络的高分辨率多尺度网格表示自回归模型。

Ascend GPU

mindearth.cell.ViTKNO

ViTKNO是一个基于Koopman理论并结合Vision Transformer结构的深度学习模型。

Ascend GPU
