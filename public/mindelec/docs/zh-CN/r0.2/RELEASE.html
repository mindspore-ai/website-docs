Release Notes

MindSpore Elec 0.2.0-rc1 Release Notes

主要特性和增强

  • [STABLE] AD_FDTD: 支持端到端可微的FDTD求解。

  • [STABLE] Maxwell3D: 支持PINNs方法三维频域电磁方程的求解。

  • [STABLE] pinnFWI: 支持PINNs方法波动方程的求解。

  • [STABLE] gprMax: 支持数据驱动探地雷达电磁反演。

  • [STABLE] metasurface: 支持物理辅助对抗生成网络超表面的全息成像设计。
