mindflow.cell.FNO1DÔÉÅ

class mindflow.cell.FNO1D(in_channels, out_channels, resolution, modes, channels=20, depths=4, mlp_ratio=4, compute_dtype=mstype.float32)[source]ÔÉÅ

The 1-dimensional Fourier Neural Operator (FNO1D) contains a lifting layer, multiple Fourier layers and a decoder layer. The details can be found in Fourier neural operator for parametric partial differential equations.

Parameters

  • in_channels (int) ‚Äì The number of channels in the input space.

  • out_channels (int) ‚Äì The number of channels in the output space.

  • resolution (int) ‚Äì The spatial resolution of the input.

  • modes (int) ‚Äì The number of low-frequency components to keep.

  • channels (int) ‚Äì The number of channels after dimension lifting of the input. Default: 20.

  • depths (int) ‚Äì The number of FNO layers. Default: 4.

  • mlp_ratio (int) ‚Äì The number of channels lifting ratio of the decoder layer. Default: 4.

  • compute_dtype (dtype.Number) ‚Äì The computation type of dense. Default mstype.float16. Should be mstype.float32 or mstype.float16. mstype.float32 is recommended for the GPU backend, mstype.float16 is recommended for the Ascend backend.

Inputs:

  • x (Tensor) - Tensor of shape \((batch\_size, resolution, input\_dims)\).

Outputs:

Tensor, the output of this FNO network.

  • output (Tensor) -Tensor of shape \((batch\_size, resolution, output\_dims)\).

Raises

  • TypeError ‚Äì If in_channels is not an int.

  • TypeError ‚Äì If out_channels is not an int.

  • TypeError ‚Äì If resolution is not an int.

  • TypeError ‚Äì If modes is not an int.

  • ValueError ‚Äì If modes is less than 1.

Supported Platforms:

Ascend GPU

Examples

>>> import numpy as np
>>> from mindspore.common.initializer import initializer, Normal
>>> from mindflow.cell.neural_operators import FNO1D
>>> B, W, C = 32,1024,1
>>> input_ = initializer(Normal(), [B, W, C])
>>> net = FNO1D(in_channels=1, out_channels=1, resolution=64, modes=12)
>>> output = net(input_)
>>> print(output.shape)
(32, 1024, 1)
