# Release Notes

## MindSpore Insight 2.1.0 Release Notes

### 主要特性和增强

#### Profiler

- [STABLE] Profiler支持收集Host侧各个阶段耗时数据。
- [Beta] Profiler支持收集Host侧各个阶段内存数据。
- [Beta] Profiler支持收集数据处理算子的执行耗时。
