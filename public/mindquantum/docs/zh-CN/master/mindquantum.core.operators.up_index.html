mindquantum.core.operators.up_index¶

mindquantum.core.operators.up_index(index)[源代码]¶

索引顺序，默认情况下，我们将无自旋轨道设置为偶数-奇数-偶数-奇数（0,1,2,3,…）。 自旋向上的自旋轨道（α轨道）索引为偶数。

参数：

  • index (int) - 空间轨道索引。

返回：

int，关联的自旋向上的轨道的索引序号。

样例：

>>> from mindquantum.core.operators import up_index
>>> up_index(1)
2
