mindquantum.dtype.is_single_precision

mindquantum.dtype.is_single_precision(dtype)[源代码]

判断一个类型是不是单精度类型。

参数：

  • dtype (mindquantum.dtype) - MindQuantum 支持的类型。

样例：

>>> from mindquantum import is_single_precision, complex128
>>> is_single_precision(complex128)
False
