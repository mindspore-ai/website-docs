mindquantum.framework

量子神经网络算子和cell。

Operators

接口名

概述

支持平台

mindquantum.framework.MQOps

包含encoder和ansatz线路的量子线路演化算子，算子返回在参数化量子线路（PQC）演化出的量子态上哈密顿量期望值。

GPU, CPU

mindquantum.framework.MQN2Ops

包含encoder和ansatz线路的量子线路演化算子，算子返回在参数化量子线路（PQC）演化出的量子态上哈密顿量期望绝对值的平方。

GPU, CPU

mindquantum.framework.MQAnsatzOnlyOps

仅包含ansatz线路的量子线路演化算子。

GPU, CPU

mindquantum.framework.MQN2AnsatzOnlyOps

仅包含ansatz线路的量子线路演化算子，算子返回在参数化量子线路（PQC）演化出的量子态上哈密顿量期望值绝对值的平方。

GPU, CPU

mindquantum.framework.MQEncoderOnlyOps

仅包含encoder线路的量子线路演化算子。

GPU, CPU

mindquantum.framework.MQN2EncoderOnlyOps

仅包含encoder线路的量子线路演化算子，算子返回参数化量子线路（PQC）演化出的量子态上哈密顿量期望绝对值的平方。

GPU, CPU

mindquantum.framework.QRamVecOps

QRam 算子，该算子可以直接将经典数据编码为全振幅量子态。

GPU, CPU

Layer

接口名

概述

支持平台

mindquantum.framework.MQAnsatzOnlyLayer

仅包含ansatz线路的量子神经网络，ansatz线路的参数是可训练的参数。

GPU, CPU

mindquantum.framework.MQN2AnsatzOnlyLayer

仅包含ansatz线路的量子神经网络，ansatz线路的参数是可训练的参数，网络输出测量值的模方。

GPU, CPU

mindquantum.framework.MQLayer

包含encoder线路和ansatz线路的量子神经网络，encoder线路将经典数据编码成量子态，ansatz线路的参数是可训练的参数。

GPU, CPU

mindquantum.framework.MQN2Layer

包含encoder线路和ansatz线路的量子神经网络，encoder线路将经典数据编码成量子态，ansatz线路的参数是可训练的参数，网络输出测量值的模方。

GPU, CPU

mindquantum.framework.QRamVecLayer

包含qram和ansatz线路的量子神经网络，qram将经典数据直接编码成量子态，ansatz线路的参数是可训练的参数。

GPU, CPU
