mindsponge.common.rots_scale

mindsponge.common.rots_scale(rot, scale)[source]

Scaling of rotation matrixs.

\[\begin{split}\begin{split} &rot=(xx,xy,xz,yx,yy,yz,zx,zy,zz) \\ &scaled\_{rots} = (scale*xx,scale*xy,scale*xz,scale*yx,scale*yy,scale*yz,scale*zx,scale*zy,scale*zz) \end{split}\end{split}\]
Parameters

  • rot (Tuple) – Rots, length is 9, \((xx,xy,xz,yx,yy,yz,zx,zy,zz)\) . Data type is scalar or Tensor with the same shape.

  • scale (float) – Value of scale.

Returns

Tuple, scaled rotation matrixs. Length is 9, shape is the same as the input rots’ shape.

Supported Platforms:

Ascend GPU CPU

Examples

>>> import numpy as np
>>> from mindspore import Tensor
>>> from mindspore import dtype as mstype
>>> from mindsponge.common.geometry import rots_scale
>>> x = Tensor(np.ones(256), mstype.float32)
