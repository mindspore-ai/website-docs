.. py:function:: sponge.function.any_none(iterable: Iterable)

    如果可迭代对象中的ANY值 x 为 None，则返回 True。

    参数：
        - **iterable** (Iterable) - 可迭代变量

    返回：
        bool。any。如果可迭代对象中的任一值 x 为 None。
