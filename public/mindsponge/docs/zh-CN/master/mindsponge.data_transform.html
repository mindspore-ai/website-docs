接口名

概述

支持平台

mindsponge.data_transform.atom37_to_frames

根据8个刚性组计算每个氨基酸的局部坐标系，返回shape为 \([N_{res}, 8, 12]\)， 其中8表示每个残基依据原子对扭转角的依赖可分成至多8个刚性组，。

Ascend GPU CPU

mindsponge.data_transform.atom37_to_torsion_angles

计算每个残基的7个扭转角并且以正弦、余弦编码。

Ascend GPU CPU
