mindsponge.cell¶

接口名

概述

支持平台

mindsponge.cell.Attention

多头注意力机制，具体实现请参考 Attention is all you need 。

Ascend GPU

mindsponge.cell.GlobalAttention

global gated自注意力机制，具体实现请参考 Highly accurate protein structure prediction with AlphaFold 。

Ascend GPU

mindsponge.cell.InvariantPointAttention

该模块用于更新序列表示（即输入inputs_1d），在序列表示中加入位置信息。

Ascend GPU

mindsponge.cell.MSAColumnAttention

MSA列注意力层。

Ascend GPU

mindsponge.cell.MSAColumnGlobalAttention

MSA列全局注意力层。

Ascend GPU

mindsponge.cell.MSARowAttentionWithPairBias

MSA行注意力层。

Ascend GPU

mindsponge.cell.OuterProductMean

通过外积平均计算输入Tensor（act）在第二维上的相关性，得到的相关性可以用于更新相关特征（如Pair特征）。

Ascend GPU

mindsponge.cell.Transition

两层全连接层，中间输出特征数为输入特征数的倍数。

Ascend GPU

mindsponge.cell.TriangleAttention

三角注意力机制。

Ascend GPU

mindsponge.cell.TriangleMultiplication

三角乘法层。

Ascend GPU
