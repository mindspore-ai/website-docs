sciai.architecture.NoArgNet

class sciai.architecture.NoArgNet(backbone, *inputs)[源代码]

返回指定网络第一个输出的网络。

参数：

  • backbone (Cell) - 原始网络。

  • *inputs (tuple) - 原始网络的输入。

输出：

Union(Tensor, tuple[Tensor])，指定输入经过原始网络的结果。

支持平台：

GPU CPU Ascend

样例：

>>> import mindspore as ms
>>> from mindspore import ops, nn
>>> from sciai.architecture.basic_block import NoArgNet
>>> class Net(nn.Cell):
>>>     def __init__(self):
>>>         super().__init__()
>>>     def construct(self, x, y):
>>>         out1 = x + y
>>>         return out1.sum()
>>> net = Net()
>>> x, y = ops.ones((2, 3), ms.float32), ops.ones((2, 3), ms.float32)
>>> no_arg_cell = NoArgNet(net, x, y)
>>> res = no_arg_cell()
>>> print(res)
12.0
