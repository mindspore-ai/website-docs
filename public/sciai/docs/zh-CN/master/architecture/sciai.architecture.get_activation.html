sciai.architecture.get_activation

sciai.architecture.get_activation(activation)[源代码]

根据名字获取对应激活函数。

参数：

  • activation (str) - 激活函数的名字。

返回：

Callable，对应的激活函数。

支持平台：

Ascend GPU CPU

样例：

>>> from sciai.architecture.activation import get_activation
>>> sigmoid = get_activation('sigmoid')
>>> print(sigmoid)
Sigmoid<>
