sciai.common.DatasetGenerator

class sciai.common.DatasetGenerator(*data)[源代码]

常用的数据生成器。

参数：

  • *data (any) - 用于迭代的数据。

异常：

  • TypeError - 如果输入类型不正确。

支持平台：

GPU CPU Ascend

样例：

>>> import numpy as np
>>> from sciai.common import DatasetGenerator
>>> data = np.array(range(128)).reshape(-1, 2)
>>> dg = DatasetGenerator(data)
>>> print(len(dg))
64
