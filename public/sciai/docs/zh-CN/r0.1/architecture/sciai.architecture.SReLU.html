sciai.architecture.SReLU

class sciai.architecture.SReLU[源代码]

Sin整流线性单元激活函数。逐个元素地应用sin整流线性单元函数。

输入：

  • x (Tensor) - SReLU的输入。

输出：

Tensor，shape与 x 一致的被激活的输出。

支持平台：

Ascend

样例：

>>> import numpy as np
>>> from sciai.architecture.activation import SReLU
>>> from mindspore import Tensor
>>> input_x = Tensor(np.array([[1.2, 0.1], [0.2, 3.2]], dtype=np.float32))
>>> srelu = SReLU()
>>> output = srelu(input_x)
>>> print(output)
[[0.         0.05290067]
[0.15216905 0.        ]]
